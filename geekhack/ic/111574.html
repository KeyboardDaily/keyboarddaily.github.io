IC Geekark Selenites, any suggestions/comments appreciated

Introduction:
Hi~I’m Mink. First of all, huuuge thanks to NightSakura夜桜さん. This set was created based on a set of “fonts”, which was hugely inspired by the simplicity and elegancy of hangul and katakana, and the popularity of Japanese sublegends - something more aesthetic rather than functional(and I’m pretty sure more than half of the users can’t really read hiragana). Combing it with some non-earthly elements and moony colors, I hope this selenite-themed set would create a sense of mysteriousness.

Why Geekark? :
Obviously, such a set would be too expensive to be a double-shot abs set, so it has to be dye-subed. Geekark is specialized in dye-sub keycaps, and they are the creator of some pretty popular sets such asMiTo XDA canvas and pbt versions of MT3 profiles. In case you’re interested, in the newly published 3ildcat video, Geekark BoW was included

Collab with Holyoops :
I thought it would be cool to have something more interesting than the typical rama artisans. Thus, I made the call for a 3d-metal-UFO-shaped artisan keycap for the keycap set(just like the the little spaceship on enter key), and I’m pretty sure the design will soon be published. It should be spinnable, and we will play with the lights (shine-through? emit light from below?). I will update on this.

Renders:
Huuuuuuuuuge thanks to Imperfect Link for the template, and credits to the keyboard model creators

I’m not a designer and this is my first time, so any suggestions are greatly appreciated(would katakana sublegends look good?). This set currently is only confirmed to be ran on Geekark. More vendors can be added if people are interested…

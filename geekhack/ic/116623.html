【IC】EVIL TROY 92 With numeric keypad

发布时间: 2022-03-18 09:26:43

This project originally started in 2020 but due to unsatisfaction of surface finish, the project was put on hold. This year we found a better surface finish factory and we felt like we could start it again. At the end of 2021, we updated the project, and please allow me to introduce you our Evil Troy 92

Evil Troy 92
7 degrees
Poron Gasket Mounts/ Silicon Gasket Mount (Both options)
Hotswap Only
On board Type C / Aviator Connector Optional
Default RGB Name Plate (Aviator Module Optional)
Standard Layout with Per Key RGB PCB or Supports Split Left Shift, Step caps lock and split back space with rgb PCB
Aluminum Housing with Brass Weight
Brass Insert for the back logo
Alumimnum Plate / PC Plate
Originally rendered in 2020
（At that time, we also designed a separate keycap set to match the color）






Revised version renders



Renders of revised version（Prototypes were made with this version）
Revised version will be compatible with RGB Nameplate and Aviator Plug


Revised Version Raw Prototype（The final product is subject to change）







We used two fixed methods




In the revised version, we have added a separate aviation plug （The final product shall be subject to the integrated scheme）




We have added a glass module at the logo on the back.
You can choose whether to install or not. The glass is fixed by rubber rings, which is very convenient to install and disassemble.
The foot pad has also been modified and improved. Now the foot pad is integrated with silica gel, which can be disassembled and replaced freely, which is more convenient.



We are already making new prototype based on the revision. We will update it in the post if there are new developments in the follow-up. If you have good suggestions, you can leave a message to us. Thank you.

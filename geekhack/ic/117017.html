【IC】Keychron K2 with Full Custom 75％ Aluminium Case

I have been working on a keyboard case for a while and finally, the first build is here. This is a work in progress and I’ll be sharing further updates soon.
Currently working on repositioning the left mounted USB-C port using a universal USB-C daughter board which can help move the position elsewhere and I’m confused about the best placement from community standards - (left/centre/right).
It would be great to hear your feedback and thoughts to make improvements while reaching the final build.
I’m also planning on designing the case to be compatible and versatile with other PCBs like Epomaker EP84, RK84, Keycool 84, Vortexgear Race 3, Akko 3084 and Drevo Excalibur. This way, I can switch the keyboards and yet have the same comfort case.
Current Specs:
Category: Case with USB reroute daughterboard
Typing angle: 7 degrees
Material: Aluminum
Mounting structure: Tray mount
Connection: Wired and wireless (Bluetooth)
Assembled weight: 1.7kg

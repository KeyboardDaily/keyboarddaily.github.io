ePBT Classic Yellow
IC FORM
Design
This is a very simple set that I was inspired to do after seeing Warren’s Classic Red. I’ve run it by him and he’s okay with me bringing this set to IC. The set will feature two stock GMK colors: N6 (yellow) and WS1 (white). I am getting a GMK color ring to send to Wei so he can match the ePBT colors to them.
This colorway wouldn’t make sense if it was GMK. We need that banana spacebar goodness.

Extra Banana Kit
If there is enough interest in the option, I’d like to have a mod-colored spacebar kit made that covers all sizes atleast twice. No render yet sorry. I also had that idea to include green accents to mimick less ripe bananas.

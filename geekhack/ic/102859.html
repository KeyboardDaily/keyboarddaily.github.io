Information:
The CU65 is a 65% mechanical keyboard kit.
The kit consists of five pieces; the aluminium top case, aluminium base plate, aluminium footbar, rubber foot and PCB.
The top case is high-profile and features an integrated plate. The integrated plate supports ANSI and ISO layouts with standard capslock and bottom row.
The PCB supports SMD RGB LEDs, hot-swap sockets, PCB/Screw-In stabs, PCB/Plate Mount switches and the software can be customised via QMK. There are two PCBS, one for ISO and one for ANSI.
The footbar is full length and provides support for the top case. The gradient is 8degrees.

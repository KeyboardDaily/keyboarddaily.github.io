I’ve been really trying to make DIY cables with a custom colored aviators for a while, but it’s not really economical in a few pieces.
So these are GX16 5 pin (maybe 4 pin if wanted) connectors that’re gonna be cerakoted at my local shop.
I’m aiming for $7-12 a piece at 25 MOQ, so it might be cheaper if we get more orders. I’m not sure though how it’ll work out, but we’ll see.
Id like to offer them in blue, black and maybe red. See the form for other colors as well. I’ll get color samples once I see if this project is even worth putting more time into.

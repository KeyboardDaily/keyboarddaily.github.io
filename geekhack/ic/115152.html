Dear Geekhackers,
Hereby helping my friends at the Percent Studio to post an IC thread for the Trio 75%, which is a successor of the Trio 60% series.
If you are interested in this keyboard, please consider joining the Percent studio Discord to stay tuned: https://discord.gg/n9ScZ8QK
You can also visit the Percent Studio website for more information and other projects of Percent Studio: https://percent.studio
Key features:

  • 75% form factor
  • PCB leaf-spring gasket mount with silicon jacket for the leaf-springs
  • 1u badge above the arrow cluster
  • Hot-swap or solderable PCBs
  • PC or Aluminum plate
  • USB type C daughter board
  • Typing angle: 7.5°
  • QMK/VIA PCB provided by Basic I/O Instrument

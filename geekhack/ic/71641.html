The font will be Gorton Modified, although Futura Oblique is shown above.
This set is inspired by cockpit elements of the Boeing 777 family with flight simmers in mind, although I’ve tried to make the base set appealing as well. The alpha colours are taken from the main panel, and the modifiers are taken from the glareshield. The 777 features lime backlighting for panel labels, I hope to recreate the effect with the glow-in-the-dark legends. The child deals include sets to add onto your macro keys for assigning them to MCP, EICAS and EFIS controls (I’m pretty sure PMDG supports keystroke shortcuts, not sure about other 777 developers). I have not included icon modifiers as I’m unsure what tooling SP has, and would like to keep cost to a minimum. I have not included non-standard keys, ISO kits or European language kits yet.

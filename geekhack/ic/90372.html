Hi,
Since launching this group buy ( http://caps-unlocked.com/group-buy-enjoypbt-pbt-dye-sub-iso-keycap-set/) on the capsunlocked store, I have received multiple requests from community memebers to create an ISO keycap set with more compatibility and better colours.
This is an interest check for an enjoyPBT keycap set that supports a variety of ISO layouts.
This is the current base kit:

The keycaps in black will not be produced with this set.
I have currently only included the ISO-UK keys, however I am am asking the community which other ISO layouts they would like to see included.
Colours also need to be discussed, and if you have any suggestions or preferences, please leave a comment below. The legends will be black unless there is wide support for another colour.

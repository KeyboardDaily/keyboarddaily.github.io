I’ve had this idea for a keyset for a while, and I wanted to gather opinions on it to see if I can make this happen! This idea is inspired from a very rare shoe colorway, so I thought it would be fun to incorporate it into a keyboard set. Hopefully, I was able to execute the idea properly, but this is still in its working stages, and anything can be changed really. I prefer to make it a sculpted SA profile set, but if there is a majority for a different profile, I can go with that one. I’ve created two different color options to choose from, and if there are any ideas, I will try to make those an option too! If the set seems to take off, I will be sending it off to Massdrop / PMK to sort the group buy out, so I won’t be handling any of it from there on. Any recommendations for kit addons or about the set are much appreciated! I will continue to update the thread with kit additions as soon as I figure out how to do so. Anyways, here are the mockups!
http://imgur.com/a/wgJz8

This was my main inspiration for the set

Update: I’ve laid out the basic kits and profile each key will be in. Also, I have decided to work with the 3 tone key color option rather than just the 2. If there are enough people wanting otherwise, I could make a change.
Update2: Just added Planck Kit! & Also decided to exclusively work on 3 tone key color due to popularity.

Update3: Thanks to Oobly, I now have a color corrected version for all to see a better representation of what they’re getting in the end. I also added Signature Plastics Color Chips & third option colorway.

The first batch of users received the kit and responded that the mounting was too tight and the positioning plate was too tight

Brass plates
PCB section (hot-swappable, Bluetooth wired dual-mode
support mac system, Bluetooth dual-mode function
support up to 2 devices to code, optional connection to Bluetooth devices,
default 6 keys without punch, can shortcut key conversion to full key without punch)

Software part is VIA solution
[/b]
Upper left corner indicator supports (static, fade, rainbow flashing, rainbow flowing, blinking, breathing light, etc.) brightness adjustment, speed adjustment, saturation adjustment (can be adjusted quickly via shortcut keys, no need to connect to software), power warning flashing function
Default firmware is wireless mode off indicator (can update firmware to on, auto hibernate to wake up by itself)
The studio provides subsequent firmware updates from time to time (QQ group and the official website provide downloads) Official website: Tyche75.com

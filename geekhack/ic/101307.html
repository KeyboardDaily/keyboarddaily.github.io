Initial “Interest Check” for the Tokamak keycap set. This set goes for an industrial control panel look while also attempting to increase the yellow pop by playing down the alphas and standard modifiers. The name is from a type of fusion reactor design called the tokamak. The novelty keys included use the name ZETA which was a fusion reactor design that was a precursor to the tokamak and instrumental toward furthering fusion reaction research.
I used the incredibly awesome DSA Scientific keycap set as a guide for Tokamak. The intention is to use Signature Plastics for production and go with the DSA key cap profile. I used the same kit designs as from DSA Scientific with some minor additions. Notably there is an additional “B” key in the alpha kit for the TGR Alice and there are included a 2u and 1u shift keys in the modifier kit. These seemed like minimal additions to include popular unique configurations. I also included a Mac Kit as I’m a life-long Mac user. I’m sure there are other small additions that could be included to maximize compatibility that I’m not aware of so please sound off below to let me know what I missed. This is my first GB so please be gentle.

【IC】Doppio - 80％ Custom Keyboard Kit (white color sample pics updated)

Hi guys! As CA66 is also having an IC, we would like to bring you our first TKL design to TKL lovers which we already held a GB in TW and CN. Doppio.

Doppio is Italian, which shares the same meaning of “Double” in English. Refers to the top dual layers visual effect we want to perform.
We tried to make Doppio simple, but look exquisite with neat lines and rounded corners. No brass weight or anything else which is luxury.
We came up with this design first on March: https://www.instagram.com/p/BgEFFRMBCDI
And then our first prototype was out in a month later: https://www.flickr.com/photos/149475414@N03/albums/72157667528136238

This version with name plate was made for modding with a stock membrane wireless keyboard.
Now we give it some changes, to make it a wire version.
Here are some renders:



We put indicators in and made a trick of them, according to the concept “layer by layer”.
And removed the name plate, engraved the Doppio name above arrow keys.
Case Features:

  • 6063 aluminum

  • Top housing (WK & WKL)

  • Bottom firm plate (PC bot plate would be an option for RGB underglows. Will make it thicker than alu one for visible from bottom)

  • Switch plate (brass plate option prob)
    Color Options:

  • Case colors: dark grey, white, grey blue, dark red, rose gold, army green

  • Switch plate colors: black, silver, red, blue, gold
    PCB Features:

  • White pcb, gold pads

  • South facing led orientation

  • USB C interface

  • QMK firmware

  • WKL 7u layout (would add ISO in if needed)

  • RGB underglow

  • indicators between ESC and F1
    Price:

  • 245USD for all colors

  • (other parts options to be decided)
    We already have a schedule to make some (probably 20) for first round. If someone is interested in it, shipping would be quite fast as it’s only anodizing left.
    Price won’t be too high since we were not aiming on that but try to make it more affordable. But still need some time to make a conclusion.
    Color options are still holding (CA66 colors can be references). Just want to show this one to you guys first to have some feedbacks.
    But we really hope that we can bring white color to Doppio by using electrophoresis, here’s a sample of it:

    As always, feel free to ask anything!

WELCOME and thank you for checking out this set!
Please fill out the interest check form here!
DSA Vegas Nights is a Signature Plastics DSA project using Trashcons and features solo numerical keys and alpha mods. Originally headed by Hedgey, he graciously handed this set to me to finish. Being Vegas born myself, I absolutely loved the set as it really captured that in-your-face attitude of Vegas Neon. Well, after over a year of dragging my feet, here it is with the attention it deserves.
This set will be manufactured by Signature Plastics, using their uniform profile DSA.
This set will be using the larger Gorton alphas, as seen on DSS Sol Dark and DSA Paperwork.
DSA Vegas Nights uses these stock SP colors:
GB
YCE OAS RCS RCU BFQ

**KITTING
**
This set now has a more inclusive all in one base kit that supports large boards. I feel this is where kitting in the hobby is going, and want to ensure this is as successful and reaches as many people as possible.

This base set will include support for as many 40s as possible, while going all the way up to TKL, and also includes 5 card suite “novelty” keys.

Further split bars include support for Corne, most split 1.75u, and 3u. Note that SP currently does not support convex small bars, so these will all be DSA. There is no 6u bar because SP doesn’t have the right kind. It’s also where the ISO keys are located. Please fill out the IC form if you would like to see them moved!

This is a numpad that I wanted to make sure looked great on it’s own and was still able to be identified as Vegas Nights, hence it having its own gradient.

**QANDA
**
Sooooo what’s next?
Next we are waiting on quotes and an appropriate time to run the set! Please follow along the thread as we get updates!
Will this have Minivan support?
No. (yes)
Will you be including support for alternative layouts?
Unfortunately likely not since with the gradient many keys would have to change.
Will the novelties from the old IC come back?
Probably not. I fell in love with the idea of doing the card suites since they can easily be plopped in anywhere, and are just subtle enough to stay on theme but add some variety. I hope you guys like them too!
Deskmats or other collaborations?
Maybe! I’ve got a few ideas just gotta get them going or get them comissioned. As far as collabs, yea I’m down, please contact me if that’s something you would like to do!
Why Trashcons?
This is something I had to wrangle with myself a bit. I’m not the biggest fan of icon mods, but I really feel like Trashcons and full icon mods lend themselves to the neon glow that this set is aiming for.
The pricing was available at the beginning for Historic, where is it here?
Since I want to have more than an NA vendor, pricing gets a little tricky. Expect it early on in the IC still, but I wasn’t able to price it at the very start with this one.

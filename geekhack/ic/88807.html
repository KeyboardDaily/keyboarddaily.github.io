Base set (105 keys)
ANSI 104 layout
Alpha base color: GFW, Alpha legend color: NN
Modifier base color: NN, Modifier legend color: RA
Includes an extra R1 1u ESCAPE - Base color: RA, Legend color: NN
Additional keys (22 keys)
R1: 1u DELETE, 1u END, 1u Page Down
R2: 1u -
R3: 1u 0, 1u +, 1u PAGE UP, 1.5u ALT ×2, 1.5u CTRL ×2, 1.75u Stepped CAPS LOCK, 7u Spacebar, 1u Blanks ×3
R4:1u PAGE DOWN, 1.75u SHIFT, 1u Blank, 2.25u “SHIFT”
Base color: NN, Legend color: RA
R3: 6.25 Spacebar, 7u Spacebar
Base color: GFW
Pantone references:
Warm Grey alphas - Pantone 402C
Black modifiers/text - Pantone 419C
Red modifier/text - Pantone 485C
One footnote, the R3 1u blanks will be NN/NN doubleshot of the R3 Numpad period, per Maxkey there is a possibility of seeing small outline where the character would be, so I picked the smallest surface area character, the period, to minimize it.
The R4 1u Blank will be of the R4 Page Down, as it’s the only 1u R4 mold to use.

Features

﻿
Bold Iskar features a one-of-a-kind bold, mirror-polished rim. The modular nature of the rim allows for easy matching with keysets.
﻿
Customizable Multiple weight, rim, weight and sound dampening options allow for hundreds of possible combinations.
﻿
Accessible NOTE: This render shows south facing switches. The real board will feature north facing switches for compatibility. The Iskar will feature a multi-layout hotswap PCB for easy installation and even more options. Further, Iskar is able to be programmable via (pun intended) the VIA software.
﻿
Comfortable A gasket mount design allows for increased bounciness and also helps to create a more poppy sound. Flex cuts on the PCB and a neutral 7 degree typing angle allows for a bouncy and comfortable typing experience. Multiple plate options ensure a comfortable configuration to each user’s preferences.

Update 04/21/2018 -
It’s finally time everyone! GB for GMK Violet Tendencies will be opening Sunday, April 22nd and running through May 20th! Kit layouts are directly below this blurb, and are set in stone. Apologies to those of you whose changes I couldn’t implement, there was a lot of compromises that had to be made to make everyone at least a little happy. I’ll add the link to the GB page when it gets approved
Base kit - Will cover a vast array of boards; $160 at MOQ, $150 at 250 kits

Space kit - Self explanatory; $35 at MOQ, and lower as we go.

NA will be through Dixiemech https://dixiemech.com/shop/gmkviolettendencies
EU will be through mykeyboard.eu https://mykeyboard.eu/catalogue/category/group-buys/gmk-violet-tendencies_46/
Asia will be through zFrontier

Update 8/28/17 -
We’ve got an EU proxy set up. Which means faster and cheaper shipping for you folks. 40/45% kit will now include 8 extra (2 for each row) 1u keys, maybe with the proper text mods on them, maybe blank, waiting on pricing quotes. Will also include a 2u purple and a 2u gr1 for the ortho folks. Kit renders (and fixes) will be done soon!
Update 12/4/17 - It seems my estimated GB will not be February as I’d hoped. I failed to discuss the time frame with the vendor that is helping me with it, and he’s picked a different set to help run before mine. Seeing as how that set will be somewhat similar, I’m going to have to put mine off for some time, so as not to flood the market with too many purple themed keysets. To conclude; this set will still happen, but it’s going to be put on the back burner for a while. Apologies to everyone who’s been eagerly following me, and checking in weekly.
Update 01/26/2018 - Not much of an update, but for the sake of transparency (of which is sometimes lacking in our hobby), I’ll keep y’all up to date. Not much has progressed. I’ve not had any luck finding someone to do renders that I can afford (if y’all know someone, send em my way ). The vendor that was chosen seems to be full up for another 4 months at least, so I’m not certain if we can still go that route. I’d really like to, but this needs to happen in 2018, preferably the first half of 2018. So there we are! Not a good update I realize, but, as I keep saying, this will happen! Hope everyone had a good holiday, and has started a happier new year!
Update 03/28/2018 - New kit design is done. New vendor has been chosen. I’m happy to announce that Dixiemech (of GMK 9009 fame) will be running the GB. Thank you all for following through all this! We’ll very soon have estimated launch dates, and official renders and pricing. For obvious reasons, I’m not going to speculate. I guess all I can say (again) is stay tuned! It’s finally happening!

Ok guys, quick update, and not necessarily the good kind.
The darkest color (GSF) with the current PBT SA tooling texture basically turns the caps into small chalkboards. For the various test runs, SP’s production crew were EXTREMELY careful and took extra time to ensure the keys weren’t scratched, but even if the added labor was affordable, the second those keys hit the shipping bags… The set would be completely ruined…
So we are adjusting the coloring a bit, and restarting the overall tooling process. This is going to add a few more weeks onto the production time, but GRADE is coming this summer, one way or the other. I will post up new imagery as soon as I can update it, along with the updated color codes. Below is the cellphone snap SP sent me of the existing (old) colors.

When I originally planned GRADE, I included ISO and non-standard keys in with the Base Kit so we could help our brethren and sisters out with MOQ’s. However, since this is a Set of the Month, MOQ’s aren’t a thing. So, Melissa at SP and I decided to break the kits out completely, this should make the whole set even MORE affordable for MORE people, which was the whole point of one set in the first place.
Pricing will break down as follows:

  • MANUFACTURING HAS STARTED!

  • 7u Spacebar and ISO Enter:

  • Both will be PBT!!!

  • The ISO Enter key will be blank, SP is unable to print on the curved surface of that cap at the moment. I figured the community would rather have a blank PBT cap vs an ABS cap.

  • Escape kit:

  • We are going to break the escape keys out of the main set, so anyone who purchased Ice Cap or Industrial (or any other R3 set for that matter) can get these.

  • I added an extra red key so people with other sets can enjoy the full spectrum.

  • I also added in a violet cap to both round out the rainbow, and also enable the kit to be used across the FN row.

  • Multiple Colored Kits:

  • There have been a few requests for something along the lines of colored mods, or arrow keys. We are holding off on these for now.

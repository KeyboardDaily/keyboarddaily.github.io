【IC】GMK Delta - Starting Nov 1st!

Hey everyone,
I’d like to reintroduce GMK Delta! This is a design by Emil.
It features WS2, CR, and the VBV Teal from Signature Plastics. This is the same teal used in Hyperfuse Origins, Hyperfuse Remix, and Tokyo Nights.

So unfortunately, Emil has been really busy recently, so I’ve essentially taken over running GMK Delta.
He is very grateful that the set will still run, and I still want to stay true to his vision. But the old thread was confusing, so I’ve tried to collect all the most up to date information here.
**>> I would love to get your feedback and gauge interest in this form! <<
**
You can also come talk to me on my Discord
Or keep up with my Instagram
The kits are as follows:

Base Kit
The base kit includes an extra TAB key so you’ll have a full set of extra mods to mix and match. These mods would pair really well with hyperfuse origins, tokyo nights, hyperfuse remix, etc. along with WoB/BoW sets.
The base kit includes 4 unconventional 1U keys: R2 “Out”, R3 “Data”, R4 “Shift”, and R4 “Enter”. This means with a single base kit, you will have support for building out an ortho keyboard with a 2U spacebar. If you want a sculpted spacebar for ortho, you can get the spacebar kit.
This kit will be $134.99 on CannonKeys

Space - Spacebar kit
This kit will be $27.99 on CannonKeys

Deskmat
This mat will be $16.99 on CannonKeys

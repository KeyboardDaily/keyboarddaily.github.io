EPBT Hemkund

Happy Diwali Everyone!
This is EPBT Hemkund
Theme music
Please consider playing this while you read the IC
The goal of the set was to bring a unique sublegend to keycaps which is rarely used, Gurmukhi. Gurmukhi is the written form of Punjabi. Additionally the layout is inspired by the Gurmukhi Inscript layout.
The inspiration for the set comes from the Sikh Temple of Hemkund Sahib, This temple was built in the late 1600s to early 1700s. The temple itself is found deep in India and within the Himalayan mountains, and is surrounded by mountain peaks on all sides. You can see one of the reference images below, along with an explanation of the colour choices made.

More

  • Darkish green was chosen to represent the surrounding grass and flora
  • Bluish White for the snow and the roof
  • Blue for the pond/lake, aquamarine is pretty ugly so i wanted a more clear blueish color
  • Beige for the stone

    Regarding the sublegend placements you can learn more below
    More

The sublegend placements were done by referencing the 2 layouts below, in combination with the input for Gurmukhi found on different OS if there are any issues or corrections please do let me know and I can make changes immediately. Some legends were omitted, due to busyness and some repeats.

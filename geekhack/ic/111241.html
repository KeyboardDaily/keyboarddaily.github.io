【IC】Ilex - Split Ergo with exposed components

Ilex is a 44 key split ergo with exposed components, optional dual rotary encoders, OLEDS, and reversible TRRS jacks. The current version of the PCB supports LED underglow, Kailh Hotswap sockets on the North face, and Through-Hole Soldering on the South Face.
The layout is a 5x3 staggered column, a three key thumb cluster, a deconstructed pinkie column with an angled mod, and two optional 128x32 OLED displays. Rotary encoders are also optional on the key south of the OLED. 44 Keys total, 42 with both encoders.
There are two options for placement of the TRRS jacks: The first option is on the top edge facing north, the other is on the inside edge just south of the encoder. This provides more customization options for cable placement. Currently, there is only MX switch support, however Choc may be available in future if there is enough demand.
Firmware is now working in QMK.

The layout is still being tweaked as using it over time shows various inefficiencies and improvements to its mapping for common uses.

The layout is heavily influenced by the Jian layout, which is an extremely efficient mapping based around using both Latin and Cyrillic alphabets with a limited number of keys.

Hello everyone!
I would like to introduce a set I have been working on for quite awhile
The inspiration for this keycap set was a post-apocalyptic wasteland full of toxins, poisonous gas, and hazardous materials.
Instead of going for more of a chaotic theme for these keycaps, I decided to give it a more modern and clean touch.
The result of these ideas gave way to my latest creation…
xPBT POISON

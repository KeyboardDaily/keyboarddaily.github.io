Here’s the official group buy: http://geekhack.org/index.php?topic=63953.0
Here’s the original group buy for GMK Cyan modifier keyset, but we’ve decided turning it into full set. http://geekhack.org/index.php?topic=62048.0
We’re done with most keys, just need some help with ISO keys. For now, we’re planning to change the R3 | key to R4 which fits right next to 1.25 left shift. The other keys should remain as is.
We’re planning to provide three sets and two extra keys.
Set /#1: Base set

Set /#2: LED add-on set

Set /#3: `~ and | keys in cyan in order to have a balanced color matching.
Key 1: Blank Cyan R4 (bottom row) 1.25 Key
Key 2: Blank Cyan R4 1.00 key
MOQ for the base set is 50, the MOQ for the other keysets and keys is 150 (this 150 MOQ will be shared with a group buy in China, so it shouldn’t be that hard to reach).
The group buy price for the base set is around $120 depending on if we’re going to add any keys to this set.
Please post here if you are interested, or have any questions or suggestions.

Introducing- GMK Bread

Updates 01-07-20
Group buy is now live! We’ve moved over here!

Important Updates 29-06-2020
GroupBuy Dates have been confirmed to be 1st July-31st July! We have also added DeskHero as a Canadian proxy for the set
Pricing for the set is as follows
Base Set - 134.99 USD
Alt Alphas - 83.99 USD
Extensions - 39.99 USD
Novelties - 42.99 USD
Deskmat - 16.99 USD
Collaborations for the set
Currently, were looking at custom cables, a clay sculpts, and a Rama keycap for the set.
Unfortunately, with the vendors wanting to start the GB, some miscommunication between Louis and myself about the GB date (apparently Early July meant July 1st and confirmation was not given to me before they announced), and with some supplies still not in from these makers, were still waiting on samples (with the exception of the Rama keycap of course, which Ive been working with Rama to get a confirmation out ASAP). Nonetheless, the GB will not be delayed and we will sort everything out.
Cables are made by @fatboychard
Im working with another local artisan maker for the sculpts! Will be a fun little surprise.
Also, to prepare for the Group Buy, I’ve done a quick signature for the set! Feel free to use it to show support!

Oh and, hop onto the discord server!
Server link: https://discord.gg/VujDfB

Important updates 18-06-20
We have gotten confirmation from GMK regarding the price. Vendors are now working everything out!
In the meantime, I have made a discord server for the set, and hopefully other sets that I might have in the works! See you there!
Server link: https://discord.gg/VujDfB
Frankly, Im not too sure how I got this idea. It just, sort of clicked one day and wouldnt you know it, I came up with this beauty. GMK Bread features a relaxed brown colour way, which remains pretty rare in this community.

Good news everyone! Today we will be delivering to the keycap set KAT Slurp to the mechanical keyboard enthusiast community of Planet Earth.
This set is inspired from Futurama, and the best-selling drink of the universe: Slurm. This highly addictive set represents the luminescent soda encased in a dark blue can with the brand name Slurm written in bright pink. I hope you enjoy this set as much as I enjoyed creating it.

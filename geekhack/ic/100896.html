6. FAQ (PLEASE READ BEFORE ASKING QUESTIONS)
Why this layout?
I wanted to provide a similar amount of functionality that a TKL would bring but in a more compact form. Everything apart from the main alpha cluster has been separated to be able to navigate it easier but still be within easy reach. The top right corner has been blocked because I found that you had to move your hand a lot to be able to reach it (I would be using my index finger for this key). All separations are 0.25u for consistency.
Will there be a WKL blocker version?
I probably won’t be adding this option for a few reasons. First is that the design is catered towards my personal preferences and I am a WK user and I don’t want to deal with an SKU for the machining and lastly, I don’t like the look of uneven blockers (Dixie did good with the Bauer layout).
Will there be a WKL 7u layout or 2x3u splitspace?
The layout below was considered but I opted not to include it because it messed up the footprints on the PCB due to the overlaps and I only want to provide one plate.
Likewise for splitspace - the 2 inner stab holes overlap the centre switch mount area on the PCB.

How about ISO?
As with the above, I’ve opted not to include ISO support because of how it messes with the footprints for ANSI and as selfish as it sounds, this a design that is tailored to my personal preferences and is optimised for that.
Will hotswap be available?
Sorry, only one type of PCB will be available due to its complexity and it would also split quantities which would affect pricing.
Why the gap between the top and bottom?
It is to avoid a standard butt-seam. In architecture, if there is a change in material, there is usually something to mediate between it which we call a negative detail or we’d use a trim. Also, I was aiming for a detail where the shadow line merges into the cutout for the USB port as per the below.

Will there be per key RGB or backlighting?
There won’t be any RGB or backlighting provided. The only way to get backlighting would be to solder your own in-switch LEDS.
Can you do some more renders?
I stopped doing renders after deciding on which colours I want to offer. If you are wanting renders done of your keyset, you can ask Abec who I’ve sent render files of the 7V to.
Do you have a discord server?
I don’t have the time to maintain a server but there is an unofficial channel dedicated to the 7V on the Daily Clack discord server here

【IC】GH60 PCB with split long keys and arrows, full LED, buttom LED

类型: IC
作者: sennin32
发布时间: 2017-05-04 13:07:58
更新时间: 2017-05-05 04:03:21

Hi, guys, we’re Doyu Studio and I’m sennin32.
Some geeks might know that we stop offer Satan PCB months ago.
Mainly because the packing quality.
We spend too many time to fix before they can ship.
Even we change supplier, the quality is horrible.
So, we plan for making a replacement PCB recently.
Instead of basic GH60 PCB, we tried to add some new ////.
For me, keyboards without arrow is not a good option for daily use.
I were a soft engineer before, without arrow is okay, but arrow is important for me now.
If we want to add arrow, there’s some thing we have to change - Right shift.
So we’re thinking about split space bar into 2 key.
There’s some requirement of this design:

  1. Contain arrows
  2. Keep Shift on the right hand
  3. Don’t change alpha layout
  4. Keycap must easy to get
  5. Micor USB, cable is much easier to find
  6. Support as many as possible of existing GH60 parts
    The layout support :
    http://www.keyboard-layout-editor.com//#/gists/a1bef8bf2784f5d009c96665a19bfc5e
    This idea is mainly come from minila, so, the space bar has the same size as minila.
    This is the PCB design now.

    One disadvantage is one of the screw won’t able to use because of add keys.
    (The screw near spacebar)
    PCB design is almost finished, we’re now checking if there’s any bug and making optimization.
    The 3u space’s stabilizer mounting hole is not yet finished.
    We haven’t confirm correct dimension with factory.
    First question is will you guys interesting in this layout?
    Or do you have any idea for the layout?
    Because of the new layout.
    We have to make a new firmware before we start massive manufacturing.
    Means we’ll have some sample PCB for design new firmware file and testing.
    Since making 1 PCB and making 10 PCB has almost the same cost.
    (Actually, cost for SMD soldering for 1 Pcs is 1/2 of making 1k Pcs…, I mean, total cost)
    We might make around 10 samples.
    Give away around 5~8 Pcs for reviewing and testing purpose.
    But have no idea how to chose guys we should send now.
    Second question is could you give us some idea of how to chose who we should give sample to?
    This PCB might spend around extra 2 months before manufacturing.
    So this GB won’t start soon O_Q.
    The other small issues:
    Will you interesting in costar stabilizer and costar plate?
    (They won’t be universal, only support several layout, so price will higher since we have to making different plate support different layout)
    Interesting in PCB GB with keycaps?

GMK Void
Good Afternoon Geekhack,
While working on Deepwell I have come to the decision that it is best to make the Void kit it’s own project as a lot of people liked it and wanted more compatibility for the kit so now I introduce to you, GMK Void!
A spin off set from GMK Deepwell! We’ll start at the end point of it from GMK Deepwell and work towards a more improved set based on feedback and reccomendations.

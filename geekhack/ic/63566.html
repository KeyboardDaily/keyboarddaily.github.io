GON NerD MobiK PCB:
USB Full NKRO & 6KRO(Toggle)
1000Hz Polling Rate
Fancy LED Mods: Backlight LEDs and Side LEDs which are controlled independantly(256 scale Brightness and Varous Effect)
Easiest and most Powerfull H/W Macro(Total Key Inputs of about 2,000, 1~70 macros can be stored, Subtle adjustment of delays between key inputs)
Supported Key Layout: ANSI Only(Centered Caps L or Off-Centered Caps Lock)
Three user definable key layers
Multimedia Keys(USB Only)
Optional Bluetooth Connection(USB/Wireless Combo): Ver2.0 or later only
Controller SMD Parts Soldered and Firmware Flashed before Shipping(Diodes, Resistors and etc are not presoldered.)
NerDy GUI Software Offered(for Windows, Full Detailed Configurations)
Command Line Interface Offered(for Linux, LEDs configuration and Simple Macro Managements only)
This set includes:
GON NS NerD Crystal MobiK Housing /w AL anodized bolts, CNC-machined Polycarbonate and Diffusing finish
New NerD MobiK White PCB supporting FB155BC_S Wireless Module
New MX Clears or Blacks(One should be chosen.)
A set of Stabilizers for MobiK KBD
GON Diodes(Very easy to solder on NerD PCBs)
RED Switch Stickers
470R Resistors 50pcs for In-Switch Backlight LED or On-PCB Backlight LED
150R Resistors 50pcs for Side LED
Optional FB155BC_S Wireless Module + KSD02S Slide Switch + Li-Pol Battery /w PCM + GON Charger Circuit Parts
What you further need for better customization(Most of them can be bought with the DIY set):
Optional Switch Springs
DIP LEDs for In-Switch Backlight LEDs
SMD LEDs for Side LED(50pcs) or for On-PCB Backlight LED(50pcs)
Lub for Switch Slider - Krytox 103 and 107(Mix them to get your favorite viscousity)
Mini-USB Cable

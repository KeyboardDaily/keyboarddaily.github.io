https://forms.gle/Cvzs5t53VCi45C3H7
Description:
Axon is a 40% keyboard built using only through-hole components, similar to and inspired by many other keyboards including Plaid, Gingham, Romeo, Discipline, etc.
Axon is very compact at only 12u wide, is QMK compatible, and has a USB-C connector.
This is a continuation from my post on r/MechanicalKeyboards:
https://www.reddit.com/r/MechanicalKeyboards/comments/n0fhxn/axon_my_diy_40_throughhole_keyboard/?utm_source=share&utm_medium=web2x&context=3
where many commentors expressed their interest in a group buy.
I am willing to run a group buy if there is indeed sufficient interest, so please fill out the IC form if you are interested!
Axon is open hardware; you can find the designs on my github: https://github.com/thatfellarobin/axon.
Of course, this means that you can produce Axon for yourself without going through a GB process for a kit, but the GB would save you the effort of sourcing parts and will lower the cost if all you want is a single unit.
Details:

  • An illustrated build guide will be created.
  • The kits will ship from Ontario, Canada.

Changes for Axon v1.1 (GB version):
Note: this change list is not finalized

  • Fixed issue where the stabilizer blocked a mounting hole.
  • Spacebar widths can be swapped (i.e. 2.25u-2.75u or 2.75u-2.25u).
  • Update screw placements to ensure acrylic cover symmetry along both x and y axis.
  • Optimize decoupling capacitor placement.
  • Polyfuse now has room to lay horizontally if desired.
  • Minor adjustments to back side silkscreen.
  • Numerous minor layout and routing changes.

To be included in the kit:

  • Top and bottom PCB
  • Acrylic component cover
  • Standoffs and fasteners
  • All necessary electrical components
  • ATmega328p with bootloader installed

Not included:

  • Soldering tools (soldering iron, solder, flux, etc.)
  • Other tools (screwdriver, side cutters, etc.)
  • USB-C cable
  • Keyswitches
  • Keycaps

Regarding group buys and trust:
I will be fully transparent: I have never run a group buy before, and have never sold product online. However, I believe that this group buy can be successful for the following reasons:

  • The available quantity will be limited to ensure that I do not end up with an unmanageable number of orders.
  • This kit does not involve substantial manufacturing risk. The parts are not difficult to procure, and I will source the PCBs and other components from reputable sources.
  • The online store will be run as a Shopify store or similarly reputable online commerce platform in order to make sure that payments and your data are secure.

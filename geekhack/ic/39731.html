【IC】g80-1800 & g80-186x compatibility set for SP keycaps

Time to-do an interest check for a g80-1800 compatibility set to be used with SP keycap sets.
Typically with the combination of base kit + 10key kit + tsangan kit you would have enough caps to fit everywhere on an 1800 board, but it had the wrong profile and/or legend for 5-6 of them.
If you were trying to fit an 1863/1865 board, you needed to order 2 tsangan kits to get the 7 1x keys and find a 6x spacebar from elsewhere to use
To make a set fit all keys on an 1800 with the correct profile, you need 5 or 6 additional keys. This will be called the 1800 kit
-SP ROW 4 1x 0 key (this key is usually included with the 10key kits, but for WYSE sets it doesn’t exist anywhere)
-SP ROW 3 1x +
-SP ROW 2 1x -
-SP ROW 1 1x Delete
-SP ROW 1 1x End
-SP ROW 1 1x Page Down
To make a set fit all keys on an 1863/1865 with correct profile you need 8 additional keys. This will be called the 186x kit
-SP ROW 4 1x Ctrl(qty 2)
-SP ROW 4 1x Alt(qty 2)
-SP ROW 4 1x windows(qty 2)
-SP ROW 4 1x meta(qty 1)
-SP ROW 4 6x spacebar
Here is a picture showing the keys necessary for an 1800 in redand the additional keys necessary for an 186x in purple:

anything else like the 1.75x right shift or ISO things are covered already in the tsangan kits or ISO kits and not here.
I don’t expect there to be a lot of interest for these, so even hitting the usual 25 minimum tier will be expensive. So no pricing as of yet
I expect there will be even less interest in the 186x kit, which is why I have things split out in 2 at least for the interest check.

Calice - A 75% F13 Alice

The ‘Complete’ Alice.
IC Form | Discord

R4 Prototype
Background
Heya Geekhack, for the longest time as an Alice user I always wanted something that I could use on the daily for work, but never found something that exactly met my needs in an Alice layout (F-Row and Arrows). The focus was to create an Alice with a function row and arrows while trying to keep the form-factor as close as possible to a normal Alice layout in width.
Features

  • Case Angle: 5.0°

  • Case Material: 6063 Aluminium & Sandblasted Brass Weight

  • Weight: R1 Proto 2.8kg unbuilt, R2 Proto 3.1kg unbuilt(est.)

  • Colours: TBD (poll in IC form)

  • Plate: 1.5mm Aluminium (FR4, POM, PC, CF, PP as potential extras, TBD)

  • Mounting: Geonworks Tadpole + O-ring

  • Feet: SKUF Feet

  • PCB: Custom PCB designed by Jels (MX Compatible, Encoder Support, QMK and VIA Compatible)

  • Alphas Tilt: 10.0°

  • Front Ledge: 18mm

  • Optional Wrist Rest

  • Layout -Layout supports normal/stepped caps, 2U+1.25U Left Spacebar and Split Backspace and Rotary Encoder at Home key

【IC】Siso keyset (on hold)

May 5, 2015 IC, 键帽 本文总阅读量次

类型: IC
作者: MAR82
发布时间: 2015-05-05 10:38:09
更新时间: 2015-12-13 16:28:49
原链接: https://geekhack.org/index.php?topic=71629.0

I have had this idea for a keyset in my head for some time. So I just had to get it out there to ask what people think and see if there could be some interest in such a set.


I was inspired by other sets, like the most obvious would be Soware http://keypuller.com/soware/, or Calm Depths http://keypuller.com/calm-depths/, and also the blue accents from the Dolch GMK https://cdn.shopify.com/s/files/1/0490/7329/products/Dolch_Addon.JPG
The set would use Cherry style legends (sorry if my pictures doesn’t use the right font or arrows)
I would love to have the set made with Cherry profile caps, but I’m not ready to run a GMK GB by myself.
The other problem with having it be GMK would be kits. I would like to make some additional kits for different/less common layouts.
SP is the best for the ease of creating a GB and adding additional kits, but I don’t really want this set with DCS caps (and I don’t think there are many fans of DCS anyway)
Some of the kits I would like to make for it would be one for the blue accents (the blue keys in the pictures would all be optional), a 40% kit, numpad, non standard bottom row, and maybe an international kit
So please let me know what you think
Update 2:
The profile of the keys will be Cherry, I hope to work with the manufacture to match the colors as close as we can.
The Windows logo will hopefully be the Windows version 1.0 logo, unless not at all possible, then I will use the Windows 8 logo.
Font will be Cherry
List of different kits:

  • Base kit
  • Color kit
  • Non standard layout kit
    The non standard kit will support HHKB, 40%, and I have plans to add G80-1800 support.
    Please let me know if there’s anything people really need or want to change.
    I will post updated mocks whenever I can
    Update 1:
    Ok everyone I have made some new mockups. I personally really like the RGB mods, but I can understand that not everybody does, or will want them. So I have pulled them from the base set.
    I’ve also added mockups for a number pad, ISO kit, Tsangan kit, 40% kit, RGB kit, and a kit with blue accents.
    As of now the colors being use are those used by Signature Plastics with ABS plastic.
    Dark grey = GSZ
    Light grey = GTA
    Blue = BBQ
    As I said before I would really like this set to be produced in Cherry profile, but have no plans on running a GMK GB myself (but if someone wants to handle that part, let me know).
    greath said that I should maybe ask Bunnylake to see if he could possibly be interested in running it in the future GB with his new keycap partner JTKeycaps. But before I do that I would like to have everything all set and ready.
    If Cherry profile is not an option I’d still like to have the set made in SP DCS profile since it’s the closest we would be able to get to Cherry. But I might also hold a poll for that, if we get there.

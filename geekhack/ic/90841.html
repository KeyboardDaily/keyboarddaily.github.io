Ok, so this IC is starting off light on content, just so I can get a feel for if this is even worth it. I really, really want some Aurebesh keys and they’ve either been done in a way I don’t like (Milestone) or they’re taking forever (REBEL). So, I had an idea: all that is needed for a DSA set, really, is to provide the TKL Base Set ( https://pimpmykeyboard.com/dsa-sublimated-keycap-sets/) from SP and any modifier kits can be ordered directly from them. That way, only one kit has been designed, there will be no (less?) worries about hitting MOQ, and people can choose the mod color that they think goes best. Unless the response is overwhelmingly negative here, I’m going to start working up a layout and I will update this post with it. If you have any comments or criticisms, please, don’t hold back!

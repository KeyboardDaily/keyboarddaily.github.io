There are a lot of ICs, so I need your help to keep this up to date. If you want to help out, contribute to the google sheet which contains all the data for this tracker.
Link to Spreadsheet - Sorry for link shortener but the google sheets link contained a homophobic slur which was starred out by the spam filter, breaking the link.
Anyone can edit this document. Please help to keep it vandalism clear. Just add each IC as a row, filling in the appropriate fields.
Current ICs Only - if the part has gone in to production please request that it is added to the Group Buy Tracker.
If there has been no progress on an IC for at least a month, then please do not add it to the spreadsheet. Chances are that it is dead or dormant.
TYPE should be one of the following: KEYCAP, KEYSET, KEYBOARD, ACCESSORY, PART
KEYCAP is for single caps or small sets (GH labelled keys for example).
KEYSET is for whole keysets
KEYBOARD is for a keyboard or keyboard kit such as LightSaver
ACCESSORY is for a keypuller or keyring and the like.
PART is for switches, springs, landing pads etc.
If this is a totally new IC, specify it as Round 1
Thanks!

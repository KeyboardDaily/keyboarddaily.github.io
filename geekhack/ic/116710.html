Hello everyone! Paraluman is my take on the gummy o-ring mount boards. Top mount is also an option for those who don’t like the o-ring mount. We already ran 2 batches of GB for this board locally here in the Philippines. A prototype run (5 units) and a mini gb for local enthusiasts (30 units). This year, we would like to offer it to everyone!
Kit Details:

  • Aluminum Top and Bottom Case, and Plate
  • WKL top case by default but can offer standard top if there is demand
  • Center USB-C using daughterboard
  • Solder PCB
  • External Brass Weight (3mm Thick)
  • Mounting: Top Mount or Gummy Oring Mount
  • Front Height: 20.5mm
  • Typing Angle: 7 degrees
  • Includes a 50A NBR O-ring (default), 30A O-ring can be an option if people prefer that (I just need to find a supplier for it)
    More Photos: https://imgur.com/a/zaaLLH8

Price and GB Schedule:
This is still TBD. We would like to gauge interest first before asking for updated quotes from the manufacturer. Prices have increased from the GB run to the mini GB so we also expect increase in prices this year. Our target price is $400 and lower
Colors:
The number of colors that will be available will depend on the number of units will be ordered. So far, we have 5 colors that we have used in our previous GBs. Dual color is possible (Different color for top case, plate, and bottom case)

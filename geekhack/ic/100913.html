CU80

The CU80 is a tenkeyless (TKL/80%) mechanical keyboard that can be purchased either as a kit, or as a fully assembled keyboard with Cherry MX switches and keycaps.
This keyboard features an anodised aluminium case that is made of two parts; the base and the top case. The kit is only available in grey. The kit features a hotswappable PCB that available with either the ANSI layout, or the ISO layout. The PCB supports QMK and features SMD RGB LEDs under each switch. The software can be heavily customised meaning that you can change key locations, have multiple Fn layers and more! You really can make this keyboard your keyboard. The inclusion of a hotswappable PCB means that there is no soldering. Want to change switches or have different switches in different locations? No problem, all you need to do is take out the current switches and place the new ones. This PCB is available with either the ANSI or ISO layout. The PCB supports screw-in stabilisers and PCB mounted switches.The PCB supports USB-C.
Why did we design the CU80?
When we started designing the CU80 we had a few criteria; it had to be high-quality, feature-rich and be affordable. We feel that this mechanical keyboard fulfills those criteria and is a great option for those looking to get a custom mechanical keyboard but cant accommodate a $300+ budget.

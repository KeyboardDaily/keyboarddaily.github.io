【IC】SA Oblivion V2 - Launches August 2020 on Drop.com

March 28, 2020 IC, 键帽, SA 本文总阅读量次

类型: IC
作者: Oblotzky
发布时间: 2020-03-28 16:44:23
更新时间: 2020-08-03 15:43:56
原链接: https://geekhack.org/index.php?topic=105348.0


Hello everyone,
I’d like to introduce the second version of SA Oblivion. I’ve been asked a lot through PM’s and such when this set would return, after all it has soon been 2.5 years since the last time you could purchase this without having to scour extras and mechmarket, so it’s long overdue. Originally, my plan was to run the set again in MT3 doubleshot when I saw matt3o doing MT3 Susuwatari. However that set has yet to ship, and also I would want to use a different font for this (and other) projects if I were to go the MT3 route. So at this point this would still have taken until next year to get into group buy. Therefor, I see no reason to hold back this set any longer to return in high profile.
Initially, I just wanted to trim a bit of fat off the old version, and rearrange some kits and call it a day. But once I finished drawing up 28 kits, I realized this was just too many these days. Firstly, I feel that SA sales have declined, SA Carbon R2 was the pinnacle with SA Pulse R3 following strongly after. Every SA set since then has been doing OK but not as well as you’d want them to do to support that many kits. Additionally, COVID-19 and its effects on the economy cannot be ignored, so I expect that to be another reason why sales won’t be through the roof despite the current 7000+ requests sitting on the Drop page. Lastly, when you look at the extras still available at Drop ( screenshot for archival purposes), a clear pattern emerges for what kits vendors end up sitting on after doing the favor of buying them out to MOQ.
For these reasons, I have removed some decorative kits (tenkeyless.asm, macros.asm, LED, OSX, communities), as well as Minila, Colevrak and ISO layout kits. Once we’ve weathered 2020 and MT3 Oblivion happens, I intend to bring back Typist and International support.
So onwards with the new stuff! Below you will find my first revision of the new kits. Some changes I’ve had in mind are adding a modifier colored F13 key to the Alternate Function Colors, as well as replacing the Assembly/40bit kits with a combined Git version similar to what GMK Monokai Material will be offering, but I’m not sure if that kit would end up being too expensive having to get new legends made again. Not sure where to put a second B, I guess in the Alphas kit would be the best place, despite looking like an error to most people.
Oh and also I’ve added ASCII alphas as seen on GMK ASCII. I’ve chosen not to have the keys with tertiary ASCII legends like they are present in GMK ASCII because those select few keys would need their primary and secondary legends shifted to the left which would look odd when everything else stays centered. And even when shifted, the ASCII sublegend would have to be smaller than all other ASCII sublegends to fit on the remaining space, which again would look very odd.
This group buy will be hosted by Drop this Summer.
Please voice your interest and offer feedback! Thank you,

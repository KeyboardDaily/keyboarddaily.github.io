The Set
It’s summertime! Who doesn’t love sitting down, getting a nice sunburn, and enjoying some refreshing watermelon? This set is inspired by my favorite summer snack.

This set uses CP as the alpha color, and two custom RAL colors to capture my interpretation of a delicious watermelon.
Kits
GMK Watermelon will have 3 kits, Base kit providing basic ISO coverage, as well as numpad coverage, a cute novelties kit, and a basic spacebar kit.

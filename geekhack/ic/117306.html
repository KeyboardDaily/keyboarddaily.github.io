Hi everyone!
We are about to start a new Group Buy. It will be much appreciated if you could spend some time giving us some feedback!
Find details in IC Form
Get more updates on our Discord Server
Est. shipping date: Late July, 2022

  • A TKL gasket mount hot-swappable mechanical keyboard DIY kit!
  • A large capacity of 6000mAh battery allows the keyboard in dual-mode to have steady transmission with bluetooth and type-C connetion, as well as supports triple-mode plus 2.4Ghz connection (a specific dongle needs to be purchased).
  • Structured gasket-mount with Everglide screw-in stabilizers, the keyboard is offering a stable and consistant typing flow.
  • The RGB backlights are shine-through not only from the sockets but also underglow.
    Specs:
    Model: Epomaker Everglide SK87 Gasket DIY kit (switches and keycaps not included)
  • Number of Keys: 87 keys
  • Size: standard TKL
  • Structure: Gasket-mount
  • Hot-swap Availability: 3/5-pin Hot-swappable
  • Connectivity: USB-C/Bluetooth
  • Case Material: CNC Acrylic Case
  • Stabilizers Type: Screw-in
  • Backlight: Per-key & underglow RGB
    Some are questioning if this is a Groupbuy - well, we would like to talk about the scene behind it. Some might have known Everglide and their switches and stabilizers. And they also have released their first keyboard in 65% - Everglide SK68. We did try out Everglide SK68 and were amazed by everything of this board, so we teamed up with Everglide to make this TKL keyboard/DIY kit. It is not around the market yet as we are not sure if TKL is still a thing because, man, 75% is just too trendy recently. That’s why we wanted to start this GB. Please don’t mock if it’s a north-facing one because it’s kindof a habit of manufactuers making keyboards like this. So we still need more voices to negotiate with them to modify this board. So feel free to leave your opinion in our IC Form!
    The ETA shipping date is just for small amounts, as we are planning to make some for the first few backers who would like to support this GB, so that the GB will not be ceased because of limited backers.
    Again, thank you for all your valuable views about this!

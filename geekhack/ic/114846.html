WELCOME TOMIAMI!
A keycap set inspired by that one show all the 80s kids watched.
Before I continue, I’d like to address a few things first:
-As many of you know, no, this is not my original design. This set is based on the GMK Miami and GMK Miami Nights colorways. I decided to run this set after I and another designer, KucherenkoZZZ were both told on separate occasions by Originative Co and GMK that we would not be able to re-run Miami Nights, even though Originative is not the original designer of the colorway. But I still wanted to bring it back into the community because these colorways haven’t been ran in years. So, if GMK won’t let you run it, then what do you do? You switch profiles/manufacturers.
-Why KAT? Well, because… why not? One of the main reasons I’m choosing to run with KAT is due to the way Keyreative does their MOQ system, where a certain amount of total keys has to be ordered rather than a certain amount of kits. This favors sets with a larger number of kits, which allows for wider compatability across different layouts and regions.
-The timeline. Yes, I know, it is notoriously difficult to communicate with Keyreative. I don’t even have a vague timeline as of now, especially not without input from Vendors, but my goal is to have this set in groupbuy before the start of Q2 2022.

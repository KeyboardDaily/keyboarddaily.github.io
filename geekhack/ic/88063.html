    Summary Feature Set for GH-122.2017
  1. Mega size with a 7x24 maximum layout (up to 156 key positions!).
  2. Full programability.
  3. Alps compatibility for expanded ANSI only.
  4. Flexible layout beyond just ANSI/ISO to include functional subsets like 5x24 "Wide Body", F-122 layout – including the bent F-key rows, etc.
  5. 100% through-hole constructions, so no advanced equipment or skills are needed to successfully build it.
  6. Mounting holes for 5x24 build.
  7. Added lower mounting holes to PC-122 upper bank.
  8. Daughterboard mounts for the controller.
  9. Added center-line support holes between Row B and C (matching GH60.)
  10. Jumper blocks for upper bank and LEDs, located and labeled for use in the Unicomp case.
  11. Alps compatibility for expanded ANSI only.
  12. Modular/removable keypad.
  13. Split spacebar support.
  14. Partial backlighting circuits, especially transistor mounting locations.
  15. Addressable RGB LED strip circuit, e.g. to attach WS2812B. (Firmware NOT included!)

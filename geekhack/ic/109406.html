Introducing the GetawayVan

IC Form 2:https://forms.gle/84qgN98UcmLjgdWx7
https://forms.gle/ac7dY78GttzaFwf47
I got tired of waiting for van pcbs so may as well make it myself, therefore the GetawayVan is a drop-in minivan compatible pcb that features:
-Allpcb Manufacturing as they do my silks oh so pretty
-An ARM microcontroller
-Usb Type C
-ESD Protection
-Per key addressable rgb
-Alps support
-Solderable for those that dislike the current hotswap options
-multiple layouts for those that want split bars, big bars or a symmetrical look
Currently with Campsite in the horizon I have no plans to do an Aluminum case but I do certainly want to support it once plate files are available as this board is what will end up going there for sure. However I do plan to sell some 3d printed cases just like what I run daily as well as shill P3D’s minivan case as to not overwhelm my own printing capacity.
The board has been my daily driver for the last few weeks and overall I feel confident in that if you want a van layout now this should be what you are looking at.

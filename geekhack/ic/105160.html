【IC】GMK Monokai Material - Launches September 4th

March 16, 2020 IC, 键帽, GMK 本文总阅读量次

类型: IC
作者: Oblotzky
发布时间: 2020-03-16 12:29:12
更新时间: 2020-08-30 17:54:12
原链接: https://geekhack.org/index.php?topic=105160.0

Hello,
I have a new IDE themed keyset for you. Monokai is very popular, but I personally never really liked the standard warm version, it’s too dolchy for me. I prefer colder grays, so after a lot of experimenting and rendering I opted for the Material version of the theme. Additionally, I am making this one with a single base color unlike Oblivion/Penumbra/Dracula/Solarized Dark. Lastly, seeing the Git Base Kit in Oblivion V2 being most successful, I chose to have those as the default legends for this run. I really don’t want to offer multiple Base Kits in the first appearance of this set, but I will very strongly consider running a Regular Base Kit by itself once this has shipped after a successful group buy, so a Round 1.5 I guess.
The colors will be matched to RAL tones. I will pursue having the base color matched and finalized before the launch of the group buy as this color is vital for the set to come out great. The RAL’s I am currently eyeing for this are Stahlblau and Tiefseeblau. The other colors should be much easier to match so I will do those between GB closing and production start as is the norm with most GMK keysets.
Kit wise, I want to focus on the basics. There’s a Base Kit that covers everything essential and the 40bit kit for Ortholinear/40%/Split keyboard users. The 40bit kit is still Work in Progress because I need to go through all the layouts again and make sure the color/legend distribution works well on everything. The color distribution will be different especially for the Ortholinear layouts compared to how I did it on Oblivion V2, and the current kit forces arrows be used on the bottom row for those. I am not sure if I want to add Colevrak/International kits.

    Background
    Red Alert has run various times throughout the years. It first ran during 2011 as OTD Red Alert thanks to sixty and uranium, who worked extremely hard with SP to perfect their custom legends almost a decade ago. The set has run several times thereafter in both OG / Classic Beige and red-on-white variations.
    The CP/U9 version of this set has not run since 2015, and Id like to help make this set available to a new wave of keeb fans (including myself).

    Notable changes from the last GMK run:

  • P3 stock red instead of V1

  • R5 sculpted bottom row (and interested in adding R0 if there’s demand)

  • base kit has modern physical ISO support on R3/R4, and heavily considering adding UK ISO support to the base kit - please share feedback if you’d like this

  • no LED indicator caps, although I can add these back in if you’re interested

  • good support for 40s and Alice layouts in base kit
    Questions & Answers
    Why are you using P3 instead of V1?
    The set last ran with V1, which looks surprisingly orange in person. P3 was introduced in the following year (2016), so as far as I know the set has never had the option of running with P3. See below for samples of three stock GMK reds.

    Will you offer a 40s kit?
    The Extension kit provides a decent amount of coverage for 40s layouts, and I am exploring how to improve coverage.
    What about aluminum artisans?
    I plan to offer aluminum artisans, but it will not be white-on-red like the RAMA options for GMK Classic Red. If you like that design, check out that GB before August 1st!
    Arent you also running GMK Birch?
    The IC for GMK Birch is indeed live. I feel confident in my ability to manage a second Interest Check, including one with only stock colors and no new original content. I also want to improve the sets availability without sending folks to Mechmarket.

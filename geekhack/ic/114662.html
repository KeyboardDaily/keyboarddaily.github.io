A keyset inspired by some of the first portable laptops.
Im very excited to post my Interest Check for GMK 1520!
GMK 1520 will use UV printing as well as double shot ABS, attempting to create a set that is currently relatively unconventional.
Inspiration
This set was inspired by some of the first clamshell laptops that paved the way for the ultralight laptops we have today.

**Updated Kitting
**
I’ve made some changes to the kitting including:

  • moving the extra black F11 through F13 to the extension kit
  • adding a 1.5u Fn novelty to base, although the design for that will change to fit the key soon.
  • adding support for the regular HHKB bottom row to the extension kit.

GMK MONOCHROME
I want to address this immediatley, there has been an idea floating around in the community for a set with black mods and white alphas for some time it seems.
Looks like both myself and JSaintS have come to similar ideas with different themes and purposes. I would like to point out that this is not a copy of GMK Empire, as we are using a different white, icon mods, and there are no novelties to match a theme (star wars) for example.
I thought I would post this to judge the interest on the colourway, but also to see how people find ws2.
However, after messing around with this combo in KLE, and working on the the set with my friend Diego (duckboi) (who I thank for helping me progress this) I think this is a set that is long overdue.
About the set:
The main design vision behind this set is to create a clean, simple, high contrast set that isnt too expensive, and has basic compatibility.
This set uses WS2 as the white and CR as the black, and Icon modifiers with a text bottom row. The use of WS2 and CR together create a high contrast, yet clean set. This set is the first to use WS2.
You may not be familiar with WS2, as it is fairly new and as previously stated, has yet to be used in a keyset.
WS2 is a very bright, cool white colour. WS1 Is child’s play compared to this.

Layout Support:
This set supports standard ANSI/ISO fullsize, TKL, 60%, 65%, 75%, HHKB, 1800, 66 key, 96 key
FAQ:
Q: What kind of homing keys will this support?
A: The set includes both scooped and barred F & J keys.
Q: What are the vendors for this set?
A: The vendors for this set havent been decided yet.
We would like to choose Novelkeys or Dixie Mech (USA), Mykeyboard EU (EU) and zFrontier (Asia).
Q: What will the pricing be like?
A: We are waiting to hear back from GMK for pricing.
Q: Why is ISO in the extras kit?
A: We have noticed that, for the most part, the majority of mechanical keyboard enthusiasts do not use ISO.
By taking these keys out of the base we can lower pricing for everyone else.
However, we arent opposed to combining the kits, or moving ISO to the base kit, if it does not increase the price by too much.
Q: Why Icon mods?
A: We think this looks cleanest and fits the vision of the set the best.
Q:Why are there no novelties?
A: Novelties are extra cost and this does not fit the design of a ‘simple’ set

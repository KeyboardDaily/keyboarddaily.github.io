This is an early-stage interest check for the Neo City Red keycap set. Do you like this style? PBT with a mix of dyesub and multi-color front-print legends. Illustrated with SP colors RBD (red), BFE (blue), YAF (yellow), VCE (green), and NQ (black), and DSA profile. Fonts are Orbitron (letters, arrows) and Roboto Mono (everything else). If there is sufficient interest, I would want this to run through MD, PMK, and/or other established shop, as I don’t want to manage GB money or logistics.
Full-Size board with TKL Base and Numpad Kit

Keyset packs, note that the TKL Base supports some split layouts (anything else you want included?)

One possible Ergodox layout using TKL Base and Ergodox Kit

One possible Planck layout using TKL Base and Planck Kit

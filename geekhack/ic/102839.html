Hi all!
This IC is for the WRISTBREAKER TKL.
The WRISTBREAKER is a high angle board which takes inspiration from several classic kustoms such as the OTD 456GT, TX1800v2, Duck, TGR Jane, etc. This was at first just a fun little meme project that I didn’t think would go anywhere, but once I got the prototype I was really happy with how it turned out. Although not THE heaviest board around, this thing is a hefty beast.
This will be run and fulfilled by goddream
Plate:

Specs

  • 14° angle

  • 1.5mm plate

  • 2x 10mm weights

  • Aluminum case

  • RGB underglow, two frosted acrylic diffusers on the underside

  • USB-C (with daughterboard) compatible with a87 PCBs

  • USB port parallel to desk

  • Fixed ANSI WKL layout (will provide plate file)

  • Compatible with TGR Jane V2 Plates

  • 9.26lbs/4.2kg built
    Other technical info:

  • 18.8mm front height

  • 7.8mm bezel height

  • .5mm switch cutout fillets

  • 14mm top bezel, 10mm bottom bezel, 6mm side bezels

  • 19.05mm switch spacing

  • 9.5mm from top of plate to bottom case

  • 8x M3 5mm (case)

  • 7x M3 5mm (plate)

  • 4x M8 16mm countersink (weights)

  • 4x M3 4mm countersink (diffusers)
    Differences between v1 and v2 proto:

  • Larger weight 8mm to 10mm thick

  • Modified daughterboard/JST cable cutout

  • Modified plate cuts

  • Various minor internal case changes

Introduction
Hey all, this is tzoyo512 from Besides Studio. After month of preparation, we would like to introduce 2 new projects to the custom keeb community. First project will be a full metal key caps set proxy by Thocc Supply (by Mr. Nobody) and second project is our height adjustable 75% metal keyboard kit named M·ONE75% (R2)
M·ONE 75% is a height adjustable full aluminum keyboard feature a standard ANSI 75% layout. The project itself began in June 2019 and completed its first limited run of 100 units in China in December 2020. We made quite a number of improvements over the first run and now it is ready for the international custom community.

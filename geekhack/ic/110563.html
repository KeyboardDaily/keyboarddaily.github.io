KAT Love

This is a short update on KAT Love.
As you may know Keyreative is not in a great place right now with regards to production. I am not as hopeful as I was before on their prospects, so we have decided to postpone KAT Love for the time being as we search for a new keycap manufacturer. The set will still run but we would like to finalize the new manufacturer and update kitting before announcing anything further.
Hey everyone, this is KAT Love

Kits and Board Renders
Kits
We are looking to offer 4 base kits with this set, the primary purpose for this being to create kitting that is beginner friendly and provides a simplified purchasing experience.
We will still be offering standard KAT kitting which allows you to mix and match kits to get what you want most.
Each base kit supports supports UK ISO, Ergo, and a number of modifier options for 60%-TKL (including F13); 1 base kit and 1 set of alphas should cover 2 board a 60/65 and a tkl.
For this set I chose not to use the standard KAT font, The fonts being used are Roboto Medium, and the sublegends and Japanese use Noto Sans JP Medium

GMK Swiss addon kit
GB Post is Live!
During my previous project I got some requests to add a Swiss layout, but due to the nicheness I didn’t want to offer it in a custom color, so this is my answer!
This is a simple IC for a GMK addon kit to finally add the Swiss layout as an option for custom keyboards.
==============================
What is the Swiss Layout?
Due to Switzerland having 4 national languages, most keyboards have a funky layout which lets them use 1 set of keycaps for all the languages. This is that layout!

Kits:

This allows for both an ISO and ANSI version, Side legends are UV printed.
Inspiration:

The Letter positioning was based on this old Cherry G81-3077 which had Dye sub Keycaps.

Colors:

WS1 for the Caps and CR for the Legends for BoW and reverse for WoB. These match a lot of keysets, regardless of color!

UPDATE - 8/13/21 - Group buy dates: August 25 - September 25, 2021, now with (some) prices
We are less than 2 weeks out from Group Buy, and pricing on keycap kits from some vendors is now available! Pricing from other vendors, as well as for deskmats and metal artisan collabs, will be added/updated as they become available. You’ll also see that some vendors are offering the Fledgling Bundle, which is a chance to get all of the kits (other than ISO from these particular regional vendors) for a discount!

Besides additional pricing, still to come are one more metal artisan collab and the 1stGarden deskmat design! More updates to come before group buy!
Join my Discord for first looks and sneak peeks, as well as new board renders! I’ll keep this post, Instagram, and the SA Fledgling website updated regularly. However, there will be informal leaks, etc. posted on the Discord, so check it out if you’re interested (there are also cute cat emotes).
Asero Foundry collaboration (8/17)

This is a 2.25u SA R3 metal keycap, for an artisan Enter keycap. It’s a bit of tongue-in-check NorCal slang, and I hope people like the playfulness of this cap combined with the understated design.
Ikki68 Aurora x Fledgling collaboration - NEW VERSION (8/13)
The Ikki68 Aurora collab has undergone a makeover! Basically, after the first iteration of the board went through IC, it was decided to make the design more unique and interesting, with an understated floral design on the back, sides, and just a bit on the front of the board, inspired by Shannie’s Fledgling deskmat design! (Thank you to summerqm for design execution and pwade for the renders!) In addition, unlike previous versions of the Ikki68 Aurora, this collab design features an e-coated alu top case for a heavier and more premium feel!

CHANGELOG
3/12 - Post IC
4/21 - Physical color reference photos shared, kit updates posted with highlights as follow:

  • TKL Mod kit has been updated to a more comprehensive 60%-TKL Mod kit, and the name of the kit has been changed accordingly. 65%/75% macro column compatibility has been added, in addition to 1x 1u CTRL and 1x 1u ALT keys in R3.
  • The keys added to the 60%-TKL Mod kit have been removed from the Extension kit. Per several IC feedback requests, a stepped CTRL key has been added to this kit.
  • In the 40s/Ortho Mod kit, 1.5u Backspace key has added (Whimsy compatibility), 1.75u R4 Function key has been added (Dimple compatibility), 1.5u Control and 1.5u Alt keys have been added, and 1.75u and 3u spacebars were removed.

4/28 - very minor kit updates, with highlights as follow:

  • Alpha-colored f-row keys moved to the Alphas kit, along with the 7u spacebar. Removing the alpha color from the 60%-TKL kit should make this kit cheaper, without any trade-off in terms of compatibility.
  • 1.5u Backspace legends across both Mods kits were changed to “BACK” for aesthetic reasons.
  • One R1 1u bird novelty was changed to a ladybird for a bit more variety.

5/11 - vendors, collabs, and more

  • Added estimated GB timeline of Q3 2021
  • Added vendors for US, CAN, EU, UK, China, SEA, and Oceania

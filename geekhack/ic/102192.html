GMK Alter is a keycap set inspired by the character Saber Alter from TYPE-MOON’s Fate media franchise.

From the RAL Classic and Design systems.

Kits/Novelties
There will be a single hiragana base kit and a separate spacekeys/bars kit for compatibility with most 60, 65, 70, 75, 660, TKL, 96, 1800, Full Size keyboards in ANSI, tsangan/WKL, ISO, HHKB configurations + more.
The novelty kit will feature a set of treasures fit for a dark-tainted tyrant.
Base kit ($130 @ 250 MOQ)

Spaces kit ($20 @ 100 MOQ)

Novelties kit ($40 @ 100 MOQ)

Deskmat
Swallow the light.
For only $20 you can empower your set of GMK Alter to maximum strength for maximum value!


Sleeves
Waterproof cloth keyboard sleeves by yasintahir. They will be available as zipper or drawstring configurations with the option of a cable pocket. Sizes available will be for TKL, 65, Full-size, 60, and 75. They will be run alongside the GMK group buy for $27**$25**.

Where to buy?
The group buy will start on November 1, 2019 and end on November 26, 2019.

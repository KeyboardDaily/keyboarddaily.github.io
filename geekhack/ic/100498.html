【IC】ADK64 3D Printed 60％ Case R2

May 2, 2019 IC, 键盘, 60 本文总阅读量次

类型: IC

No brakes on this train. I’ll be setting up this page for gathering a little more info for another round of the ADK64. Here are my plans:

  • Larger amount of spots, as 50 went pretty quickly last time
  • More colors, I kept things pretty muted for R1, I think we should get some bright colors for R2… maybe even carbon fiber filament
  • Apple Logos to go with the boards - these are hard to track down, but I’ve got some leads.
  • Potential geometry change to make printing easier (will add screenshots and a vote to see if people don’t mind it later)
  • Collaboration with /u/inachie to bring back a run of the Paladin64 for this board. I think we can add split space as well - not sure how that would be accomodated for by AEK caps though.
  • FR4 plates - Hasu has kindly open sourced his plate file for the 60% AEK layout, so I may get a batch of these made for those that want a more complete PCB + Case + Plate setup
    Any feedback/questions are welcome! This page will be updated in the coming days with more content.

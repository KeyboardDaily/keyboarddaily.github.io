【IC】Sand Keycap set

This is an interest check for the Sand set. Partially inspired by the Granite set and partially inspired by my girlfriends favorite color palette, this is a PBT DSA set to cover as many boards as possible. As such, there will be many different child buys created to cover as many layouts as possible, and to include the option of fully blank sets if desired.
The main two colors of the set will be a light and dark tan. For accent colors, there will be a Blue extra pack and Green extra pack, along with blank versions of the two.
The current mockup image shows the blank ANSI set without either of the color packs, with the green color pack, and with the blue color pack (without the colored WASD keys). Also, for the two mockups with color packs, I used a keyboard shell color that I thought matched the spirit of the set.
Material: dye-sub PBT
Profile: DSA
Typeface: Helvetica Neue
Light Tan (alphas): Pantone PMS 614
Dark Tan (mods): Pantone PMS 451
Blue (extra pack): Pantone PMS 308
Green (extra pack): Pantone PMS 3308
Sets:

  • Base set: alphas + F-keys + Esc + arrow keys + homed F and J
  • Text mods: modifiers with text legends
  • Mac mods: modifiers with Mac legends
  • Iso set: covers ISO-specific keys
  • Numpad set: numpad keys with both Enter sizes
  • Ergodox set: Ergodox keys with text legends
  • Misc mods set: non-ANSI or ISO sized mods for other keyboards
  • Multimedia modifier set: Raise volume, lower volume, mute, play, reverse, fast forward, calculator, home, dim lights, raise lights
  • Additional homing keys: homed E, H, N, T, U
  • Blank set: ANSI tenkeyless set + 2 homed alpha keys
  • Blank ISO set
  • Blank misc mods set
  • Blank numpad: Blank numpad that covers both Enter sizes
  • Blank half-Ergodox set
  • 6.25u spacebar
  • 7u spacebar
  • Green pack: Esc, Tab, ANSI Enter, ISO Enter, numpad Enter (both sizes), W, A, S, D in VS green
  • Blue pack: Esc, Tab, ANSI Enter, ISO Enter, numpad Enter (both sizes), W, A, S, D in BFD blue
  • Blank Green pack
  • Blank Blue pack
    Hope you enjoy! Leave any comments, criticisms, or suggestions below. Thanks!

【IC】SILVERLINE keycap set - seeking specific feedback

January 27, 2016 IC, 键帽 本文总阅读量次

类型: IC
作者: Wilkie
发布时间: 2016-01-27 12:32:07
更新时间: 2016-01-29 15:08:36
原链接: https://geekhack.org/index.php?topic=79124.0

This is an early-stage interest check for the Silverline keycap set. I would appreciate your feedback, especially on the specific points listed below the diagrams. The keycaps would be DSA ABS doubleshot with (custom color) metallic silver legends on black. The goal is to run this through PMK and Massdrop (anywhere else?). Please note this design intentionally uses black for all keycaps’ base color (i.e., no striped function row, no different color Esc or modifiers, etc.).
UPDATE 29 Jan 2016: SP can’t doubleshot the spacebars, so they’d have to be blank or pad printed (if made by SP). I’m leaving the lines in the diagrams, though, in case this ideal could be possible with a different manufacturer.
Version 1 diagrams, might change:

Specific feedback I am looking for (all feedback welcome):

  • What additional languages would you use if supported, and would you want additional languages together in one child deal or each as its own deal?
  • Do you want an ErgoDox kit with legends for all keycaps? Without such a kit, would you still use this on an ErgoDox by combining (black) blanks (which some might already have) with the Base kit?
  • Smaller-than-TKL users: do you want anything added to the Planck + JD40 kit? For example, I believe it currently does not fully support Atomic. If you want something added, please be precise (size, legend).
  • Do you want keycaps with windows for LEDs, such as windowed CAPS, windowed stepped CAPS, etc.? I am not yet sure how I would handle a window on a keycap with the silver square surrounding the legend, such as S2 (in the Scroll Lock position).
  • Your thoughts about WASDEF and HIJKL kits? Leave as-is, combine into one kit, reduce to just WASD?
  • Your thoughts about the Homing kit (deep dish for Colemak, Dvorak, etc.)? Leave as-is or combine with another kit, like WASDEF, HIJKL, or even Base? I don’t have experience with this, but I imagine that combining some kits might result in higher order numbers and therefore lower prices per keycap (but possibly offset by more keycaps per kit). And not combining might mean some kits can’t reach MOQ and would have to be removed.
  • Do you want support for swapping CTRL and CAPS? If so, do you want both stepped and non-stepped CTRL, or only one? If this can’t be its own kit, to which kit should it be added, Splits or Winkeyless? Any other CAPS swaps desired?
  • Your thoughts about the Communities kit? Leave as-is, add other communities, add other types of novelties (compatible with the overall design), remove?
  • I think this should be ABS, but would anyone prefer POM for both black and silver (if metallic silver POM were possible)?
    And of course, if you like Silverline, please let me know which kits you might be interested in (no commitment) so I can begin to get a slight idea of numbers.
    Thank you!

Inspiration
The colors and color organization for this set were mainly inspired by the 1995-1996 Volkswagen Harlequin and LEGO BRICKS. But as I started reaching out for feedback on this set, folks mentioned what the set reminded them of. Hopefully, the bold colors will remind you of something too.

Kits
The kiting methodology used in this set is to provide ample coverage within the base set while keep costs as low as possible. Because this set uses five different colors in a non-standard layout, I have tried to keep the base kit as slim as possible to reduce the cost of the set.
Addon kits such as NorDe, full ISO, Ortho, and maximized 40s kits are often not successful for first-time keycap designers. Because of this, I have opted not to include them. However, as a substitute, I have included basic support to these kits within the base kit.
BaseKit
Base kit features a R1 Yo key, a R4 Yo key, and support for 60, 65, 75, TKL, HHKB, Alice (with spacebars), 1800, Full size, and Minivan 40% keyboards.

Hey all
I haven’t been doing a ton with keyboards lately, but some ideas have been wondering into my mind for keycap color schemes, and I thought I’d share one of the ones I like most.
I’ve found that I personally like how a lot of dark mod colors look with a basic tan set of alphas (I’ve put some examples below). I thought that it might be cool to run a set of alphas like this with options of which mod set to choose. Not sure what profile/manufacturer would be most preferred, but I must say I’m inclined to shoot for something SP as the low MOQs would make multiple modifier kits easier.
Here are some examples I’ve thrown together quickly on KBLE (note that I’ve only made 60% examples, but I’d prefer to run this as a TKL set):
Note: All color codes are SP ABS, but I could try to find similar colors if people want GMK or SP PBT
Link to SP ABS swatches
‘Desert’ - TEA Alphas with YW mods

‘Beach’ - TEA Alphas with BO mods

‘I have no idea what to call this one’ - TEA Alphas with RAS mods

‘Forest’ (is this too close to 23 Commando?) - TEA Alphas with VCJ mods

Input I’d love:
Which mod color is your favourite?
What other mod colors would you like to see?
Assuming all the mod colors you want are offered, would you consider buying multiple mod sets along with a single alphas set?
What manufacturer/material/profile would you like?
I personally am pretty attached to the “symmetrical” mod style as I’ve presented above, but would you like the option to have the classic asymmetrical style as another option (aka tilda and tab in alpha colors)?
Also any other feedback you care to give.
Thanks everyone!

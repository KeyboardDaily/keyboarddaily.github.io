About
Presenting KAM Spooky, the essence of Halloween in keycap form!
Ive been working on this set throughout October and with Halloween coming up I figure theres no better time to put it out there. Ive always loved Halloween colors and Im working on a new board that I want a uniform profile for and thus KAM Spooky was born. I want to go with KAM profile because I prefer PBT but wanted to use the reverse dye-sub method that Keyreative offers in order to get the orange legends on black keys.
Let me know what you guys think! This is my first keyboard set so I dont totally know what Im doing so if you have any suggestions about a better way to do kits (or any other suggestions) please let me know.
Colors:
 Dark Black + Black Legends: /#000000
 Lighter Black: /#171718 (render at the top of the page has this as the black key color)
 Orange: /#f47920
 Purple: /#7349ac

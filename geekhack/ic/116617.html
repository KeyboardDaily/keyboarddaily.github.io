  • Supported Layouts*
    There will be three plate layout options, semi-fixed. All plates support split backspace and stepped caps. ISO plates may not be available from all vendors.

  • Tsangan (7u bottom row)

  • ANSI (6.25u bottom row)

  • ISO

  • If a hotswap PCB is chosen, your kit will come with a Tsangan plate which is the ONLY compatible plate with a hotswap PCB.

  • Due to physical key compatibility with the case, an ANSI plate is only an option when choosing a WK top.

  • Due to physical key compatibility with the case, WKL and MX HHKB tops will come with a Tsangan or ISO plate, both of which are compatible options.

  • ISO only supports a 7u bottom row, but is compatible with all three top options.

  • There is no ISO or 6.25u bottom row option for the hotswap PCB. Please refer to the PCB layouts below.

  • Optimized Plate, the Process, and FEA analysis*


    Using FEA (finite element analysis) we were able to measure the simulated force on each switch plate hole to see how the plate would react to this force. The maximum displacement was recorded for every switch plate hole and relief cuts were added to the plate (iteratively) to try and make the displacement equal across all switch holes. The results are measured in max displacement in mm 10^-2. See the original IRON165 IC for details on how we went through testing methods.

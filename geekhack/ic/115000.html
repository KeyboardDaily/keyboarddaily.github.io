Specifications > 3D Viewer

  • Mount: PCB Top mount (Plateless) with Silicone pins (a.k.a Tadpole)
  • Typing Angle : 8˚
  • Front Height : 17.1mm ( https://media.discordapp.net/attachments/783301255719157761/898809888535359518/unknown.png)
  • Materials
    : Top and Bottom Cases - A6063-T6
    : Weight - Brass with Silver Ceramic Coat
    : PCB - FR4 1.6mm (Designed by Yiancar)
  • Weight(Built) - 2625g

Included

  • Cases (Top and Bottom)
  • Weight
  • PCB
  • Carrying Case (1680D Fabric)
  • 10u wires (x2) and 10u Spacebars (x4, DCS profile by Signature Plastic)

Add-ons

  • PCB : 50-55 USD (TBD)
  • Tadpoles (50a / 60a / 70a / 80a)
  • Feet (Black , Silver)
  • 10u Spacebars (DCS profile from Signature Plastic)
  • 10u Wires

COLOR
Top case - Black / Navy / Burgundy / Pink / Harbor Gray
Bottom case - Silver
PICTURE

  • The Feet is pressure fit in method , material is silicone.

PCB by Yiancar Design

Mount

Tadpole Mount

  • The Tadpole is silicon material and comes in two different hardnesses, 80a is a standard and 50a available as an option.

Top Mount with screws
Supported Layout

  • Caps Lock / Number Lock / Scroll Lock / Kana (Language Toggle) / Layer are Indicated by SMD LEDs

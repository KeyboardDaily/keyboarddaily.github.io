The Dihedral 79 is a board I have been working on for a long time as my endgame - a board that is small but also does not compromise on any usability. Above you can see an image of it’s layout - which employs an Merged Ten Key design whereby the numpad and navigation block are merged together in a way similar to that seen on the CM Storm TK
Taken from my original thread:
Based on a TKL design, the D-79 puts the f-keys behind a function layer to create a slim and attractive keyboard, while merging the num-pad into the navigation cluster to enable fast number input without a full-size keyboard. The D-79 also maintains the dedicated arrow keys that make the TKL format so attractive to many users, and splits the spacebar into two to minimise space wastage and to create a better positioned backspace.
This is an IC for a Dihedral 79 Plate group buy - a planned GB which will allow anyone interested in this design to get a plate and build a board based on it.

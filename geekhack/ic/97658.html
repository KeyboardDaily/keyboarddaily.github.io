UPDATE - April 05 2019:
Hello everybody,
It’s not a good news that SP decided to raise the cost for the new legends up to $150 each.
The reply was “production found that they can make most of these new legends work, though there might need to be some slight production tweaks done to make it moldable. All the images are very detailed and some have fine dots (some of which on the map images might not mold). These would all really be subject to the $150 charge as all are going to need new grids and quite a bit of work on our end to run smoothly.”
With this unexpected finance issues, I will have to change the kits quite a bit. Some new legends remain, the others need swapping back to the Text ver. that I have already made in 88Commando GB.

Welcome to the IC for GMK Kaiju

Kaiju ( 怪獣 ) may mean “strange beast” in Japanese, but it is more commonly used in reference to large monsters. This set takes specific inspiration from the classic Japanese monster movies featuring “the big lizard”, and derives its color palette from a combination of things related to this theme. The deep green modifiers, which reflect the classic “big lizard’s” ominous coloring, gives this set a strong foundation for the fiery orange accents and legends. Both of these colors, when paired with the light cream/beige alphas give this set an intentional nod towards classic defense terminals. The novelties, which i’ve affectionately called “warnings”, allow further expansion by providing a collection of symbols that feel authentic to this theme.

Proposed Kits

BASE KIT
Numpad split was an exploration, and one that I feel ultimately does more harm than good. I’m moving Kaiju back to a standard full base setup. Some slight legend adjustments as i’ve chosen to match sublegends to those used in GMK bento.
Adjustments include:

  • added numpad
  • accent arrows
  • orange ISO enter
  • neph lock key
  • moved 2u shift into abnormalities
  • removed 1u hack key
  • removed 00 key

TRANSLATIONS KIT
One of the biggest pieces of feedback in the form was “no weeb!”, so I hear you. I’m happy to announce a Latin alphas pack for those looking for a more traditional pairing. And yes I realize that “traditionally” the sublegend kit is the addon, but with the theme of this set i feel it appropriate to keep the sublegends in the base

ABNORMALITIES KIT
Not much has changed here, the only adjustment was the moving of the 2u shift from the base kit into this one. This saves caps in the base, as well as places the 2u shift with the other caps necessary for the “china 60” layout.

WARNINGS KIT
No adjustment other than to move the accent arrows into the base

BARRIERS
No change
Desk Mat
Unfortunately due to high failure rates from the light color, only the green variant of the deskmat will be produced.

Original -

More
These kits are by no means finalized, but they do following closely to the “current” going standard of splitting out the numpad. I personally have some mixed feelings on this, but the data is proving more and more that the community as a whole finds this beneficial.


BASE KIT
Trying to keep the most common compatibility, while still keeping the set concise. Removal of the numpad cluster keeps the cost down for the majority.

DATA LOGS
The Data Logs kit adds the numpad cluster for all that quick data entry. Essential for documenting any larger than average creatures.

ABNORMALITIES
Sometimes your standard Kaiju gets some odd permutations… This should help cover that.

WARNINGS
A good early warning system in place saves lives. This’ll help you look good doin it!

BARRIERS
When all else fails, sometimes you just need a good old fashioned blockade. In some standardized sizes of course.
Desk Mats
You’re always safe… until you’re not! Two desk mat options depending on the current conditions.

ISO JTK Keycap Set
Information
This is an interest check for a (currently unamed) JTK Keycap set that, for the first time ever, would support ISO keycaps.
There will only be one kit that will include all of the keys.

This set is still in development and nothing has been confirmed.
The price has not been confirmed but is close to the expected price based on information received from JTK.

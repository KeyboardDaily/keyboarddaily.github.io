In partnership with the gifted and awesome Gondolindrim (or Gondogod), along with 159 from ProjectKeyboard, I’d like to present the Elongate board, that combines the versatility of a 40% with the aesthetic inspired on a 1800 layout.
Please provide your feedback here: https://docs.google.com/forms/d/1xOQJRbzpbH5Ga5q6eFHe2yqL55rzjZbdXeLaXQBPJl8

Introduction
The Elongate is a custom mechanical keyboard that falls in a random place of the 40-50% range. It was designed as a new take on the mini 1800 layout, combining the versatility and portability of a 40% with the ability to have that dedicated numpad cluster, with Home, Page Down/Up, End, Insert and Delete keys.

The main and unique features of the Elongate are:

  • The protruded arrow cluster, allowing for fast navigation;

  • The blocked split space, which helps accessing a second or third layer of the 40% layout – making it more comfortable and quick to use;

  • A solid midlayer that can be either polycarbonate or brass, that improves the sound profile a lot.

  • On the spacebar blocker there are also a three-LED indicator cluster, which can be used for layer indication, Caps/Num/Scroll lock and every feature QMK allows. These indicators are fully programmable and, in conjunction with SIP sockets, allow you to hotswap LED colors.

  • As if that wasn’t enough, there are three more indicators between the numpad and the alpha cluster to indicate whatever you want it to.
    All of these characteristics really make for a very pleasant typing experience. The user is able to get the full utility of the 1800CP layout in a very compact form and have enough space in the sides to use the mouse.

  • Case: Top and Bottom Polycarbonate or Aluminium (vote for colors on the link below)

  • PCB: QMK and Via compatible (see above)

  • Extras: See below for more info
    The PCB





    The PCB was designed by my good friend Gondolindrim and features RGB underglow for those sweet lights with the polycarbonate version. It’s powered by the ATMEGA32U4 microcontroller and uses a USB mini-B connector. It also features ESD protection, connector shielding, power rails filtering and a sweet, sweet Brazilian flag. All that we learned to expect from Gondo.
    We have had until now many requests to port the PCB to USBC connector, since as it stands right now it has a mini-USB connector. Please give us a feedback on the IC form!
    The PCB is QMK and VIA compatible. Since Gondo is also a big open-source hardware advocate, the ElongatePCB is also open-source! – see the GitHub repo for the files.

IC Form
I would like feedback on the color combo and idea of running it as an alpha kit (based on notes below specifically for kitting). I do not have a current indication of GMKs new pricing for a kit like this, at this time.
A simple alpha addon kit, providing coverage alongside Drops WOB kit, or Qarmaas WOB Cyrillic, to use on multiple boards.
The extra tab allows for complete coverage of a board with an alternate bottom row, while the extra pipe keys (to be added) will provide coverage with WOB Cyrillic (arabic has no subs on pipe).
Sublegends are UV Printed. UK ISO addon keys are included as well as the extra B for ergo layouts. This set has an entirely new set of sublegends

More
Old kitting, and notes to be aware of:

  • The current version is kitted out to have full compat with Drop’s WOB kit.
  • Will add Row 1, 2, 3, 4 pipe - for compat with Qarmaas WOB Cyrillic if you don’t buy from Drop. Support Qarmaa!
  • Z key has misalignment, will be fixed in render
  • Parenthesis to be swapped
  • < >, DF, CV to be swapped

*For those wondering about swapped parenthesis, etc. This should not be done. Open MS word and check. It depends on the application’s language settings, not just the OS.
If you’re using Microsoft software, it will behave in a similar fashion to SAA. If you use a non-MS editor, or are in apps like discord, it will behave like Arabic 101 subs. I am sticking to SAA. See the difference here.

*Sublegend color - V2
**
Some notes on the thought process with the legends:

  • Disconnected glyphs are being used for each sublegend, unlike SAA, which mostly uses the connected character variant. For the sake of consistency with this character version, the sub on “I” key will remain as ه
  • Symbols are repeated on number row, a nod to many people’s endgame Arabic set, SAA. This isn’t necessary but for the sake of consistency, I wanted this

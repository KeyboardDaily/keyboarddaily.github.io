First off, sorry for the really bad formatting this is my first post on geekhack(usually on reddit).
This is an interest check for DSA Moonlit Night. (might change the name later on, will update)
Here are pictures of the current render ( http://imgur.com/a/gkVRE/#MtjeFC3)
This set is specifically designed for 68% and smaller keyboard users. I also don’t intend to make this too big of a keyset unless there is a lot of interest(200+) for it.
Some of the keys have changed since I rendered the layouts, so stick to the render of the full kit for the keys included. I am considering replacing a 1u key with a novelty.
The alphas should go well with DSA Miami Dolch and DSA Lightcycle.
Pricing is going to be $110 for domestic(US) shipping included and $130 for international shipping included. MOQ of 50 sets for this price point.
I’ve contacted SP and production time from payment to shipment from SP is 12 weeks.
Strawpoll for layouts this set will support ( http://www.strawpoll.me/13730401)
Flashquark (/u/Touareg3 on reddit, not sure about geekhack) will be running the groupbuy for this if there is enough interest.
Comment or message me for any questions and/or concerns you have!
Interest check post on reddit ( https://www.reddit.com/r/MechanicalKeyboards/comments/6ueecj/ic_dsa_moonlit_night_name_subjected_to_change/)

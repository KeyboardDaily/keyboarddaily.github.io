?Binary_Monkey
3Key Macropad, Inspired by Monkey
https://gumroad.com/jkdlab/
Hello, this is JKDLAB®.

We decided to commence our first project of 2021 as BINARY_MONKEY.
Binary_Monkey has the minimum elements of a keyboard that a programmer would require.

Theoretically, you can process any project with this single keyboard, which applies to every programmer, graphic designer and OSU player.
Binary_Monkey has been designed with great details; a compact body smaller than 5cm and 42mm PCB.
Our signature option, brass weight backplate, is provided after the careful mirrored finishing.
3-key-Binary_Monkey provides 3 different modes of input which are;

  1. Binary input

  2. Expanded ASCII code binary input

  3. Key macro input which can be customized through QMK as well as your own tool
    Including Monkey_Brown (our signature colour), there are 4 total options possible which are; 1) Monkey_Brown, 2) Retro_Gray, 3) Black and Orange, and 4) Black.

As an additional option, you may order Keycaps for DSA Profile and Brass weight backplate which PVD mirror finishing has been applied.
Pre-order is going to be opened for a month, starting on 22nd January to 22nd February.

You may pre-order Binary_Monkey on our official online store.

Flygone60 Case & PCB
A simple, budget-friendly, stacked acrylic 60% keyboard. Hey everyone, over the last few months I have been designing and producing both keyboard cases and PCBs. Now I am finally opening an IC for the design I think is ready for the community. Enter the Flygone60, the case features 6 stacked acrylic layers and a POM plate.
So far I have produced 3 revisions of the case, the first revision fixed some layer “sagging” issues as well as some spacing issues for keycap clearance.
The second revision features some changes to the keyboard feet and case curves.
The 3rd and current revision features more uniform screw placement, slight changes to the plate for stabilizer cutouts, and enlargement of the USB port opening. (It was wayyyy too narrow)
All of the keyboard prototypes I have built for the case have used the Flygone60 PCB that I designed. Currently I am testing the case with other “universal” 60% PCBs to ensure the case has greater PCB compatibility. The Flygone60 PCB as similar layout support as the DZ60, it features NO underglow and will work in other “universal” 60% PCB based case designs.
I am currently planning on running the kit standalone and offer the Flygone60 PCB as an add-on, please do leave feedback regarding this if you would prefer another option.

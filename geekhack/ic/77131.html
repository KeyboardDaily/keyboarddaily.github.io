【NQAIC】 Cardboard Keycap Set

November 16, 2015 IC, 键帽 本文总阅读量次

类型: IC
作者: derezzed
发布时间: 2015-11-16 22:12:09
更新时间: 2015-11-16 22:12:09
原链接: https://geekhack.org/index.php?topic=77131.0

I dont know where the inspiration for this design came from, but this set is dedicated to the hero of the shipping world: the box. I am calling this a Not-Quite-An-Interest-Check, or an interest check for an interest check, because I dont know if anyone can produce this color and I dont know anything about vector graphics. I lack the skills to make this a viable production design and I dont know how well it would translate to an actual set. Perhaps this post belongs in the Keycaps forum instead of the Interest Check forum. But, if there is enough interest in this design, perhaps someone would like to collaborate to make it a viable production design, or pick up where I left off and take this the rest of the way. I would like to see this set in Cherry profile, but nothing is set in stone. I would prefer to not have SA produce this in DCS, as I would like thicker keycaps, although, I don’t think it would meet MOQ as a GMK set. I also have no preference regarding doubleshot ABS or dye-sub PBT.
Novelty keys include 1.25u bottom row keys (Fragile: it must be Italian), a recycle symbol for the OS key, function keys, a box symbol escape key, a Handle With Care enter key, and a This Side Up backspace key. The highlight of the novelty keys is the space bar for people who have trouble discerning the proper space bar orientation. I call it the sh/*t starter.

I also have a concept for an artisan key I call Click-in-a-Box.

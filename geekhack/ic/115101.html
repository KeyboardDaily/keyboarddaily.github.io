Welcome to the Interest Check for my first keycap set, GMK LoB (Lilac on Black). This is a very simple keycap set that came out of me realizing I really want a simple Lilac on Black set and there being none at all.
For any questions you can find me on Discord as Sax/#2295
UPDATES:

More
Quote 11/24
-Revamped katakana sublegends, I noticed several of the legends were wrong so I decided to use an existing asset for them
-Made small fixes on some errors in numpad (It was missing a barred 5 and the scoop had no scoop)
-Updated renders with katakana subs to show the new sublegends
11/02
-Removed renders with mono katakana alphas and added a render with accent space bar
10/31
-Dropped the Mono Katakana kit. Unfortunately running two alt alphas kits would very likely result in one not reaching MOQ, and the Mono Katakana set just wasn’t popular enough to risk keeping it.
-Added NorDeUK kit to further increase compatibility, one of the main goals for this set was to have as much compatibility with as few kits as possible, the base kit will maintain basic ISO support.
-Added a Row 2 escape key for the basic 40s support.
-Revamped the Space Bar kit, removed the unnecessary 1.75u space bars as well as 1 of the 1.25u space bars, added a 1u space bar and Accent color 6.25u and 7.00u space bars, also removed the non centered 6.00u space bar as it was not necessary.
Please fill out the IC form below if you are interested in this set, any feedback is appreciated.

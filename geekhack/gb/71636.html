The successor of DSA Retro is finally here! Always lovingly designed by Matt3o (me), this time proudly powered by CTRLALT.io.
The keyset is of course in SA Profile, sculpted with the following row profile:
ROW 1 - Function keys
ROW 1 - Number row
ROW 2 - QWERTY row
ROW 3 - ASDFG row
ROW 4 - ZXCVB row (with the exception of SHIFT keys that are row 3)
ROW 4 - Modifier row

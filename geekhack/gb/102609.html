Key 65 is featured with the three-layercase and dual-gasket structure. The plate mounting system is an improved version of the gasket sandwich on Leaf 60. In addition, we add gaskets between the middle case and the bottom case. There’s an optional rubber sheet between the PCB and the plate. The numerous options will offer you the most freedom to make the keyboard look and feel as you wish.
This GB is the most sophisticated one we’ve run so far. It could also be the most sophisticated GB in the keyboard history. For each of the three layers of the case, you can choose from 3 materials, 3 finishes, and 9 colors. There are two versions of weights and 6 versions of the top cases. If we count all the options available to choose, the number of possible combinations is around 400 million. We are crazy enough to run the GB in this way. Now we invite you to be a part of this historical crazy GB.

A: Very similar.

Q: How can I choose the plate materials?

Q: What is Electrophoresis Clear?
A: E-clear is a clear Electrophoresis coating. It’s clear so the part will show its original color. This is how it looks on brass parts.

    About the PCB reset button:
    If the PCB you received has a physical button, it may not work after fully assembled. It happened because the PCB supplier changed the BOM after the prototype was tested. The button will be pushed against. the bottom case so the PCB doesn’t work. We have removed the physical reset buttons for most PCBs. Only the first 40 or so PCBs still have the physical reset button. You can simply remove it with a wirecutter or soldering iron. It’s not a necessary part for the PCB (Leaf60 PCB didn’t even have the button). If it bothers you, I can issue a full refund&return.
    Update on the firmware:
    I’ve tested the VIA firmware. It works great.

  • Download QMK toolbox.

  • Download the Key65 VIA firmware (the hex file) here: https://caniusevia.com/docs/download_firmware/

  • Flash the VIA hex file using the QMK toolbox. You need to press Fn+ESC before the flashing.

  • Download VIA: https://github.com/the-via/releases/releases

  • Open VIA then it should detect your keyboard.

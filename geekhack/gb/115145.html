GMK MATSU
By Krelbit

Hi all, this set has been in fairly active development for a while, mostly progress was shown in the Switchmod Keyboards discord. This is the official announcement for it.
GMK Matsu was inspired by the art and colors on Hanafuda cards, most particularly the suit for the month of January, which depicts a crane amongst pine trees.
Hanafuda cards are most commonly used in the game Koi-Koi, and it’s close relative Go-Stop, and are where I personally obtained my interest in them.
It’s been really enlightening and fun to be able to do a deep dive into a subject that I enjoy but hadn’t been super-versed in, and it’s been really fun designing a keycap set around it all to tie it off.
The set uses all stock colors, with UV printed sublegends, so colormatching should not be a huge issue that occupies any time.
Red: V1 (or as close as it can be)
Black: N9
Yellow: N6
White: L9
The row profile layout is 1-1-2-3-4-5. I recognize that R5 adds to the overall base kit price, but I personally think it is worth it.

Double shot replacements for Cherry MX (Filco/Ducky/Cherry).
Please visit the new thread to vote for kits and
give quantities of how much you would like to order.
The following is obsolete:
Color scheme similar or equal to Round 1.
Everything centered, no arrows, nor symbols etc.[1]
Optional extra sets for
UK, NL, AT/DE/SE, FR, Colemak, Dvorak and other special layouts.
If you like a key to be added or removed,
just state in post which and where.
Set up a poll if you like some extra symbols included.[2]
Prices are not fixed.
These are maximum prices for 1 of 50 sets (Standard kit) and
1 of 25 sets (all other kits), excluding shipping.
You don’t need to order a Standard kit along your
local/alternate-layout kit. Each kit will sell separately.
1. Standard kit @ $50

  • 104 US-layout in black/grey-color-scheme with white labels,

  • cursor-keys with no arrows; Up, Down, Left, Right instead!

  • Numpad 0-9: no arrows, no text, just numbers only (in grey)

  • Control and Caps Lock (both, stepped and normal)

  • 1 1/4-keys: 2xAlt, 2xSuper, 2xControl, Meta

  • GeekHack keys in both colors

  • to please those analphabetics: 4 arrow keys are included.

  • 2. UK kit @ $7*

  • 3. AT/DE/SE kit @ $14*

  • 4. FR kit @ $13*

  • 5. NL kit @ $12*

  • 6. Colemak kit @ $5*

  • 7. Dvorak kit @ $8*

  • 8. Modifier kit @ $18*

  • Rub Out and Symbol (stepped and unstepped version)

  • 2xHyper, Meta, Compose, Function, 2xCommand, 2xOption (1 1/4 sized)

  • Backspace, Line (at Tab or |-position)

  • Spacebar in black,

  • additional modifier keys could be added but might raise the price a bit.
    (maybe this will split up into 2 modifier sets)
    [/LIST]

  • 9. Numeric kit @ $10*

  • including 0-9 with additional single-sized 0 and vertical = (for number-block, black)

  • & % ^ = ( ) @ $ \ | |\ (for top rows, black)

  • { } : \ | |\ - + (QWERTY-row, black)

  • = - + (ASDFG-row, black)

  • |\ = (ZXCVBN-row, black)

  • plus some other keys (all in black)

  • 10. Function kit @ $10*

  • F13-F24 (matching colors),

  • F1-F4, F9-F12 (in black)

  • Cut, Copy, Paste, Undo, Redo, Help, Reset (in black)

  • plus some other keys (all in black)







    I will post layout-schemes for other kits later.
    Any suggestions or comments are welcome!

[1] If you want off-center labels, or a different color (e.g. pink),
please refer to Round TWO!

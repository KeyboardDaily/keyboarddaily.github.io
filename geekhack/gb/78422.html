Weaven Keyboard Group Buy ROUND 2

Wait is over, round 2 is here


What changes:

  • More color options.

  • Winkeyless version.

  • After a quite using, we realize the universal plate while convinient in making and handling order it is suffer from the fact that the enter key is not fully supported. We decided to separate ANSI and ISO for this round. In short, people will choose ANSI or ISO plate when ordering.

  • Also we provide assembling service if needed.

  • New shipping box, fully lined with foam to make sure problem like round 1 won’t repeat.

  • What is winkeyless version?*

  • DIS:*

  • Price:*

  • Unchanged: 1 full kit (case) = $230 (winkeyless option +$20)

  • Assembling = $30 + PCB, LEDs, switch,… (depends on your choices)
    Shipping will be handled mostly by proxy, North American order will be done by Zeal (huge thanks), European ones will be done by our own, you’ll get a secondary invoice for shipping when the order is ready to go.

  • Black, Silver, Golden, Blue, Red will be base choice, no minimum quantity required.
  • Other options will need 5 order each to be made in to production. A Google doc poll will be posted here.

  • What layouts the plate support?
    The plate is universal layout with cut out that allows you to open the switch top cover without desoldering. It is also ISO compatible.

    Those colors actually requires a mixture of solutions so it was easy when it get done for one prototype, in group buy, we need to test more for consistency between each batch to make sure everyone gets the same color so right now it is not available. I’m sorry

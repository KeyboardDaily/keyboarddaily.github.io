Tsangan kits are all the rage lately, due I suppose to the influx of Korean keyboards which tend to use 1.5x modifiers. Lots of people are trying to buy used or leftover tsangan kits from recent group buys. Why not see if there is enough demand to just run some more? To be honest, I’m not confident this will get off the ground. But I’d like to give it the old college try. Interest checks are pointless, so man up and fill out the order form!
For the uninitiated: Modern, standard-layout keyboards like the Filco, Leopold, Rosewill, Poker, Quickfire Rapid, etc. use 1.25x modifier keys, a long right Shift, and a center-mount Caps Lock. A tsangan kit adapts a particular key set for use with a keyboard that uses 1.5x modifiers, short right Shift, and/or an off-center Caps Lock. Kind of like the inverse of a moogle kit. There are some gotchas to a set like this (e.g., the off-center Caps Lock). So noobs, please: if you’re unsure about something please read the thread and then ask if you don’t see your question answered.
I have five versions in mind. Chime in in the thread if you want to add a different variation. The ones that don’t get many orders will be dropped (possibly all of them!)

  • Cream Cheese & Green
  • Red Alert 2.0 (red, white, and grey versions)
  • Cherry replica (white on black)

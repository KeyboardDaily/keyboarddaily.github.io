Novelties Explained
More

Periodic Symbol : This is the symbol that can be found on the periodic table that represents the element Mercury, its molar mass, and atomic mass.
Conical / Erlenmeyer Flask : A common flask used in chemistry for experiments that can be used to heat or boil liquids, it is also used in titration.
Rhombohedron : This is the symbol used to represent the crystal structure of the element Mercury.
Altimeter : Although modern altimeters use what is known as an aneroid, Mercury was used in the basis and invention of barometers to sense pressure changes in the environment.
At higher altitude pressure decreases, and thus a change in the environment causes the aneroid to sense the altitude.
Alchemical Symbol : This was the symbol used to represent Hydrargyrum (Mercury) in alchemy. It is also the astrological symbol for the planet Mercury.
Thermometer Scale : Mercury is used in thermometers, however thermometers that contain a red liquid is alcohol to which red coloring has been added.

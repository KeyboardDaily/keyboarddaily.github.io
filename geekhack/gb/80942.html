Specs:
Producer: GMK
Place of Origin: Wernberg-Köblitz, Germany
Material: ABS
Profile: Cherry
Matching: Cherry MX keyswitch
Technology:
Most keycaps as Double-shot
Pad printing for some special keycaps
Color code and Renders:

  1. Base Kit [127 keys]

    Base color: L9 / U9
    Legend color: CR / N7 / V1
    Matching: CHERRY G80-3000 series, WKL, OEM 104 and some custom keyboards
    Tips:
    Enter key “CHERRY” characters is double-shot, Cherry icon is pad printed.
    All the Window-LED keys,Old Windows, Menu, Cherry icon are pad printed.
    Others keys are double-shot.
  2. 1800 Kit [13 keys]

    Base color: L9 / U9
    Legend color: CR
    Matching: CHERRY G80-1800 series
  3. ISO Kit [4 keys]

    Base color: L9 / U9
    Legend color: CR
    Matching: Some European keyboards
    It’s the international edition and will update the official render by GMK later.
  4. GEEKHACK kit [8 keys]

    Base color: N9 / U9 / V2 / MS1/ CC
    Legend color: V2 / V4 / V1 / CR / WS1 / TU2 / CP
    Matching: SKIDATA+, Olivetti, Red Alert, 9009/Classic Retro, Monkey 2016, Valentine, Sky Dolch, Icon Mod Pack

All orders will be invoiced in USD and payable in USD and buyer will be responsible for payment of goods, shipping costs, for all PayPal fees.

To be grateful for your big support for this Group buy , You will get extra key(s) for who order the base kit.

Story:
I noticed dark low contrast sets like Ninja, and Black-on-Black, and said, ‘Hey, there’s no low contrast white set to put on my Polycarb keyboard!’ Then I thought to myself ‘What’s low contrast and white? … A Ghost’, and so the set was born.
Colors:
Generally, this is a low contrast set, but should still be legible. The Modifiers Base color is very close to the color of the Alpha Legends. Here is a test page you can print to roughly see.
Alpha Base: White / WS2
Alpha Legends: CMYK 2 0 0 15
Mod Base: Pantone 427c
Mod Legends: CMYK 3 3 0 27
Color Guidance:
BASE: These whites are slightly cool. Worst case they will be cooler (more blue), or more neutral. They will not be warm (red / yellow). Contrast is a change of 15% Black (in CMYK) between alphas and mods, and their respective legends. I aim to keep this +- 3% Black (CMYK).
ACCENTS: These may have more variation, but should maintain the lightness and hue. The red text should be very saturated and darker.
Font: Normal Alphas provide the same clean font on the modifiers. St Croce Alphas offer a unique, stylized font for an alternate set of alphas. This font has an optical illusion so legends appear fully formed, when in fact the thin parts entirely missing. Both ‘there’ and ‘not there’, like a Ghost. The font comes from the worn away Roman tablets in St Croce Church, Italy.

Novelties:
‘Word Poetry’, is a game where there are bunch of words that you can order to make phrases. KAM Ghost has two poetry kits that replace your alphas so you can create little messages from the Ghost (or to anyone overlooking): Ire Poetry, and Poetry. Ire Poetry uses a grisly font and blood red color for a more horror-movie ghost. Poetry uses the clean font, but with more contrast than Alphas, and a few less aggressive words.
If word poetry isn’t for you, the grisly font is also available in Ire Accent.

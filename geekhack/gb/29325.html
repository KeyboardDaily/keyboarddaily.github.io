Doubleshot Cream Cheese and Green 90-Keyset
A little history on this keyset and how it came to be. Original colour scheme was by KBDMania, sadly their GB was not international so it was very hard for some of us to get this set. Once I saw the colourway I knew it was a keeper and I definitely wanted a set, sadly it’s not as easy as it sounds. The difference between this GB and the original set made by KBDMania is that theirs had 85 fully custom fonts, and because of the size of their order legend fees was not really a problem. The current GB will have the same colour scheme but the style of the font will be Signature Plastics standard font (WYSE font, if you don’t know Google image search “Wyse doubleshots”)

Base Kit
This is a 89 keyset which includes the basic 87 keys for a tenkeyless keyboard such as Filco. The space bar is the new universal space bar which fits Filco and Leopolds (Not sure on the new leopold coming out), it also comes with 2 versions of the very classic GEEK HACK key. New addition to the set is a stepped caps lock with a centre stem!!!


Tsangan Add-on Kit
This is an extension to the base set, aimed towards the base set to fit in winkeyless boards or other custom boards such as Phantom, DOX, Cherry 1800, DK Saver, 356CL and many more!
The following is included in this kit:
2 Alt (1.5x)
2 Ctrl (1.5x)
2 Star (1x)
2 Diamond (1x)
1 Stepped CapsLock (off-centre stem) Don’t ///// at me if you can’t read
1 1800 Shift (1.75x) (This is a short shift in 1.75x size for the right shift so you can fit another 1x key beside it)
1 Space bar (7x) (Cherry size space bar for winkeyless keyboards)

10keypad Add-on Kit
This is an extension to the base set, aimed towards full size 104 keyboard users (or 10keypad users, if you are a 10keypad user please make sure the keys fit), no explanation should be required.

ISO Add-on Kit
This is an extension to the base set, aimed towards ISO layout keyboard users, the JRET is blank due to the cost of a doubleshot JRET.

Additional Space Bars!
Additional space bars will be available for purchase, seeing as the space bar is the first key to shine, people normally like to just have spares as this will likely not get a reprint at a low cost.
There will be 2 styles available:

  • Universial Standard 6.25x

  • Cherry 7x

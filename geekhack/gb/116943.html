DCS Estate: GB APRIL 18th - MAY 20th

TLDR; Beige ABS DCS set with centered legends, colorful front printing, and a focus on 40s support, to pay homage to a vintage keyboard found in someone’s basement.
Source/Inspiration
I came across a vintage keyboard replacement at an estate sale. To my despair, the keycaps on this board were not MX compatible. So I decided to make this set to be able to get something as close to the source as possible. That’s it - no story, just cool old tech.
ABS DCS was chosen to minimize lead-time, while also leveraging the superb quality keycaps that Signature Plastics puts out. This keyset is designed to be as close as possible to the original source. To do so, this set is intended to be run with centered legends. Additionally, SP will use pad printing for the colored front legends/symbols. Considering current DCS lead times, we are estimating a late Q1/early Q2 delivery range.
This set uses all stock colors:
o Legends - GTA
o Caps - TGG
o Alphas front legends & FCTN front circles - BED
o ALPHA LOCK front circles - RDA
o SYS front circles - VV
o ENTER front circles - YCH

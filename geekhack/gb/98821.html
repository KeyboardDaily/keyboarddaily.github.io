Hello, friends of Geekhack,
We are excited to start the Group buy for G60. G60 is a retro-style USB/Bluetooth Dual Mode Keyboard designed by Scott from the Basic I/O Instruments (BIOI). After obtaining feedback from the GH community in a previous IC, a successful trial run private GB, and a recent interest check, Scott has optimized many fine details, and we believe G60 warrants a full run at GH.





G60 + CF Plate +Rubber Gasket + Holy Panda: Typing test!
Instructions on using VIA configurator to set keymap for G60’s dual-mode PCB
G60’s key features are as follows.

  1. The design is inspired by Cherry G80-3000 Series, the size is cut down to 60%.
  2. Aluminum 6061
  3. A variety of colors and premium color options available
  4. Top-bottom case design
  5. Compatible with both Top Mount and standard GH60 style tray mount
  6. Compatible with most PCBs based on GH60 (via an included FPC extension kit)
  7. High-performance QMK-based Bluetooth/USB Dual Mode PCB designed by BIOI
  8. Compatible with both USB-C and USB-Mini
  9. Multiple layouts compatibility
  10. Optional plate material (PC, CF, Brass) and optional 4mm thick plates with perfect switch cutouts (alu, brass)
  11. Internal brass weight to fill the battery slot if you don’t like wireless
  12. Designed by Scott from BIOI
  13. Manufactured by ALF
  14. Possiblity to add rubber gaskets to increase typing flex

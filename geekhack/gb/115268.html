Introducing GMK Dracula V2.0! Now back for a second run with new kits and accessories!
The keycap set adapts the popular color-scheme of the same name, Dracula, designed by Zeno Rocha. Contrary to what you might think based on the name, this is not a Halloween themed set per se. Instead, GMK Dracula (V2.0) focuses on the software and coding aspect of the Dracula Theme. The colors of the set are dark greyish blue tones with blue-purple-gray alphas. The iconic multicolored syntax highlighting of the Dracula Theme is included in the form of vibrant pastel and neon legends and accents.
There has never been a set with so many different options to choose from, whether you already have V1.0 or not. Check out what’s new!

Phantom PLATES (shipping)

This is the official GB, carried over from the IC of these plates. They’re $22/plate, in stainless steel with the top face finished in line graining. These plates are REQUIRED for the PCB to be mounted mounted on the case. The plate itself is compatible with Filco, PLU, and CM quickfire TKL cases. Every switch can be taken apart without desoldering, and it’s compatible with both cherry and costar stabilizers.
ANS125 - Popular standard layout of mainstream keyboards. ANSI layout with 1.25x bottom modifiers and 6.25x space bar.

ANS150 - ANSI layout with bottom modifiers in 1.5x - 1x - 1.5x and 7x space bar. Popular with Cherry or Korean customs

ISO125 - ISO layout with 1.25x bottom modifiers and 6.25x space bar.

ISO150 - ISO layout with bottom modifiers in 1.5x - 1x - 1.5x configuration and 7x space bar.

NOTE ON THE FOLLOWING PLATES: there are a few more custom ones that I’ve added. There are no keycap sets with proper legends that will fully accommodate to these 4 new ones, not even a tsangan kit from SP. Only choice you really have is to get them from WASD, which only comes in ABS and lasered legends if you want a matching set. Otherwise you’ll have to mix and match keycaps from different sets, so buy these plates with discretion. If you’re unsure, think about also picking up a more standard plate, because there won’t be a round2 for these plates.
PRF125 - Profosist’s custom layout. Extra 2 keys in the function row, split L-shift key, split backspace, short Enter + 1x, short R-shift + 1x

HHK150 - jdcarpe’s proposed layout. It’s basically a HHKB layout with the extra function row keys that are specific to phantom PCB.

WFD125 - This is my personal favorite. It’s based on ANSI, similar to a HHKB layout but has standard function row and 1.25 bottom modifiers. Easier compatibility with keycap sets.

RBD150 - Last one for our ISO fans. 1.50 modifiers, split backspace, and short R-shift with 1x function key.

There must be at least 150 total plates ordered for these price, and MOQ for each design is 15. For plates that don’t reach 15pcs: They will either be canceled or if we’re almost at 15, people from that plate’s sub-group can buy up the rest and cost is split between the subgroup. I’ll put an option in the order form that if it were be canceled, which plate you’d want the order to default to. If the 2nd preferred plate also gets canceled, you can either pick ANS125, ANS150, or cancel the whole thing. The only two design that well exceed the 15 MOQ are ANS125 and ANS150 based on the votes.
(example: You want a HHK150, the MOQ is not reached so your order will automatically default to WFD125. The WFD125 also doesn’t reach 15 MOQ, so your order automatically defaults to ANS125).

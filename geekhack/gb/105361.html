ROUND 2: 09.20.2020
WHATS NEW (AND IMPROVED)

  • Were offering TYPE-C support for both versions (Standard with underglow and full per-key RGB) of the PCBs this time. Mini USB option is still available for the folks owning the original Alice or cases that only support it.
  • Fix the tolerance issue with Durock (and probably C3) screw-in stabs at one particular position on the last rounds PCBs (please see album). The short story is we tightened the hole to provide support for clip-in stabs, but it ended up a bit too small for Durock ones. The fix does make that particular stab position incompatible with clip-ins, but we dont think its much of an issue given how popular screw-in stabs these days compared to clip-ins.
  • Added support for rotary encoders. The Standard version has rotary encoder support for all 3 keys on the far left cluster; while the En Ciel (full RGB) version only does so for the top left key due to the fact that its schematic is more complicated with all the pre-soldered per-key RGB LEDs.
  • Added the anodised aluminum option for plates. We first offered the aluminum plate option for our previous Kanu Group buy, and since things turned out nicely, weve decided to bring the option to this one.

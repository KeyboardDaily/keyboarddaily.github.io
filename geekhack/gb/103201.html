GMK Delta
is now live!
It features WS2, CR, and the VBV Teal from Signature Plastics. This is the same teal used in Hyperfuse Origins, Hyperfuse Remix, and Tokyo Nights.

Kits:
Base Kit

The base kit includes an extra TAB key so you’ll have a full set of extra mods to mix and match.
These mods would pair really well with hyperfuse origins, tokyo nights, hyperfuse remix, etc. along with WoB/BoW sets.
The base kit includes 4 unconventional 1U keys: R2 “Out”, R3 “Data”, R4 “Shift”, and R4 “Enter”.
This means with a single base kit, you will have support for building out an ortho keyboard with a 2U spacebar.
If you want a sculpted spacebar for ortho, you can get the spacebar kit.
